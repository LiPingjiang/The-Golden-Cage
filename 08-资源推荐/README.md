## 8.8 资源使用原则

1. **少即是多**：选1-2个核心资源深入使用
2. **真题为王**：任何资源都不能替代真题训练
3. **输出大于输入**：看课不如做题，做题不如复盘
4. **信息断舍离**：不要沉迷于收集资源而忘记学习
