# 第1章 考试概览

> 本章帮助你快速了解京考的全貌：谁能考、怎么考、什么时候考。

---

## 1.1 什么是京考

北京市公务员录用考试（简称"京考"）是北京市公务员局根据《公务员法》和《公务员录用规定》组织实施的北京市各级机关公务员录用考试。录用职级为一级主任科员及以下和其他相当职级层次。

---

## 1.2 报考条件

### 基本要求（以2026年度为例）

1. 具有中华人民共和国国籍
2. 18周岁以上、38周岁以下（2026年度放宽至38岁）
3. 拥护中华人民共和国宪法，拥护中国共产党领导和社会主义制度
4. 具有良好的政治素质和道德品行
5. 具有正常履行职责的身体条件和心理素质
6. 具有符合职位要求的工作能力
7. 具有大学专科及以上文化程度
8. 具备拟任职位所要求的其他资格条件

### 特别说明

- **市级机关**：主要招录具有2年以上基层工作经历的人员
- **区级及以下机关**：主要招录应届高校毕业生
- **应届生范围**：北京生源应届毕业生；北京地区高校非北京生源本科以上应届毕业生；京外高校获得校级以上"三好学生""优秀学生干部"或校级一等以上"优秀学生奖学金"的非北京生源本科以上应届毕业生

### 不得报考的情形

- 现役军人、在读非应届高校毕业生、在职公务员
- 因犯罪受过刑事处罚、被开除党籍/公职
- 被列为失信联合惩戒对象
- 公务员招考中有舞弊等严重违纪行为
- 被辞退未满5年

## 1.3 时间安排

| 环节 | 时间（以2026年度为参考） |
|------|--------------------------|
| 公告发布 | 11月上旬（约11月10日） |
| 网上报名 | 11月17日 9:00 至 11月21日 18:00 |
| 资格审查结果查询 | 11月17日 至 11月22日 |
| 网上缴费 | 11月22日 0:00 至 11月25日 24:00 |
| 打印准考证 | 考前一周 |
| 公共科目笔试 | 12月7日 |
| 专业科目考试（公安） | 12月6日下午 14:00-16:00 |
| 成绩查询 | 次年1月 |
| 面试 | 次年2-3月 |

---

## 1.4 考试科目

### 公共科目（所有人必考）

| 科目 | 题型 | 时间 | 满分 |
|------|------|------|------|
| 行政职业能力测验 | 客观题（选择题） | 120分钟 | 100分 |
| 申论 | 主观题 | 150分钟 | 100分 |

### 专业科目（部分职位）

- 公安机关人民警察职位：需加试公安专业科目

---

## 1.5 成绩计算

- **一般职位**：公共科目笔试成绩占50%，面试成绩占50%
- **专业科目职位**：面试成绩和专业科目考试成绩共占50%，专业科目一般不超过综合成绩的15%
- 所有成绩均按百分制折算

---

## 1.6 考试流程全景

```
公告发布 → 网上报名 → 资格审查 → 缴费确认 → 打印准考证
    → 笔试（行测+申论） → 成绩查询 → 资格复审
    → 面试 → 体检 → 考察 → 公示 → 录用
```
