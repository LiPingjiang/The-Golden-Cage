# 判断推理知识体系与解题技巧

> 京考判断推理占行测30%~40%比重，2025年图形推理增至15道题。掌握规律体系是关键。

---

## 一、图形推理（15题，⭐⭐⭐⭐⭐）

### 1.1 解题总思路

```
第一步：观察图形相似度
├── 图形组成相同 → 位置规律（平移/旋转/翻转）
├── 图形同中有异 → 样式规律（叠加运算）
├── 图形组成不同 → 属性规律 或 数量规律
└── 多个独立图形 → 位置关系（相离/相接/相交）
```

### 1.2 五大规律体系

#### ① 位置规律（元素组成相同时优先考虑）

| 类型 | 关注点 | 常见形式 |
|------|--------|----------|
| 平移 | 方向+距离 | 水平/竖直/顺逆时针，等距/递增 |
| 旋转 | 方向+角度 | 顺/逆时针，45°/90°/180° |
| 翻转 | 轴线方向 | 水平轴/垂直轴翻转（镜像） |

**实战技巧**：小黑块图形用"编号法"追踪单个元素移动路径。

#### ② 样式规律（图形同中有异时考虑）

| 类型 | 规则 |
|------|------|
| 直接叠加 | 保留所有线条 |
| 去同存异 | 去除相同部分，保留不同部分 |
| 去异存同 | 保留相同部分，去除不同部分 |
| 黑白叠加 | 黑+黑=白，白+白=白，黑+白=黑（或其他规则） |

**注意**：叠加可能与旋转、翻转结合考查。

#### ③ 属性规律（图形组成不同但有共性时考虑）

| 属性 | 说明 |
|------|------|
| 对称性 | 轴对称/中心对称，注意对称轴方向变化 |
| 直曲性 | 全直线/全曲线/直曲结合 |
| 开闭性 | 开放图形/封闭图形 |

#### ④ 数量规律（图形组成不同且无明显共性时考虑）

| 数什么 | 具体内容 |
|--------|----------|
| 点 | 交点数、端点数、特殊交点（十字交叉点、直曲交点） |
| 线 | 直线数、曲线数、内外线条数、**笔画数** |
| 面 | 封闭区域数、部分数 |
| 角 | 锐角、直角数量 |
| 素 | 元素种类数、元素个数 |

**一笔画判定**：奇点数为0或2的图形可一笔画完成。  
**多笔画**：笔画数 = 奇点数 ÷ 2

#### ⑤ 空间重构（立体图形）

| 类型 | 方法 |
|------|------|
| 六面体展开图 | 相邻面定位法、相对面排除法、公共边判定 |
| 立体拼合 | 体积计算、凹凸互补 |
| 截面图 | 截面形状判断 |
| 三视图 | 主视图、俯视图、侧视图对应 |

### 1.3 特征图形速查

| 图形类型 | 常考规律 |
|----------|----------|
| 汉字 | 笔画数、部分数、封闭区域数、结构（上下/左右/包围）、同一元素遍历 |
| 字母 | 封闭区域数、对称性、直曲性、间隔排列 |
| 阴影图形 | 平移、面积比、对称 |
| 箭头/指针 | 旋转方向和角度 |
| 嵌套图形 | 内外元素独立变化 |

## 二、逻辑判断（⭐⭐⭐⭐⭐）

### 2.1 翻译推理（假言命题）

| 关联词 | 翻译规则 | 逆否命题 |
|--------|----------|----------|
| 如果A，那么B | A → B | ¬B → ¬A |
| 只有A，才B | B → A | ¬A → ¬B |
| 除非A，否则B | ¬A → B（等价于 ¬B → A） | |
| A是B的充分条件 | A → B | ¬B → ¬A |
| A是B的必要条件 | B → A | ¬A → ¬B |

**推理规则**：
- ✅ 肯前肯后：A → B，A成立，则B成立
- ✅ 否后否前：A → B，¬B成立，则¬A成立
- ❌ 肯后不能肯前（无效推理）
- ❌ 否前不能否后（无效推理）

**联言与选言**：
- A且B：两者都为真才为真
- A或B：至少一个为真即为真（否定一个可推出另一个）
- 德摩根定律：¬(A且B) = ¬A或¬B；¬(A或B) = ¬A且¬B

### 2.2 削弱与加强

**论证结构**：论据 → （论证方式）→ 论点

| 类型 | 方法 | 力度排序 |
|------|------|----------|
| 削弱论点 | 直接否定论点 | 最强 |
| 削弱论据 | 否定论据真实性 | 较强 |
| 切断联系 | 论据与论点之间无关联 | 中等 |
| 另有他因 | 存在其他原因 | 较弱 |

**力度排序**：直接否定论点 > 切断联系 > 否定论据 > 另有他因

**加强**：与削弱相反，搭桥（建立联系）是最常见的加强方式。

### 2.3 真假推理

**解题步骤**：
1. 找矛盾关系（必有一真一假）
2. 确定其余命题真假
3. 回代验证

**常见矛盾**：
- A 与 ¬A
- "所有都是" 与 "有些不是"
- "所有都不是" 与 "有些是"
- A→B 与 A且¬B

### 2.4 排列组合推理

**方法**：代入排除法、列表法、假设法

## 三、定义判断（⭐⭐⭐⭐）

### 解题方法

1. **提取关键信息**：主体、客体、行为方式、目的、结果
2. **逐项验证**：对照选项检查是否满足所有要件
3. **排除法**：任一要件不符即排除

### 常见陷阱

| 陷阱 | 说明 |
|------|------|
| 主体偷换 | 选项主体与定义主体不一致 |
| 范围扩大/缩小 | 概念外延不匹配 |
| 常识干扰 | 用生活经验代替定义文本 |
| 条件缺失 | 选项缺少定义中的必要条件 |

## 四、类比推理（⭐⭐⭐⭐）

### 常见关系类型

| 大类 | 具体关系 | 示例 |
|------|----------|------|
| 语义关系 | 近义词 | 美丽：漂亮 |
| | 反义词 | 勤奋：懒惰 |
| | 象征义 | 松柏：长寿 |
| 逻辑关系 | 种属关系 | 苹果：水果 |
| | 组成关系 | 轮胎：汽车 |
| | 对应关系 | 医生：医院 |
| | 因果关系 | 努力：成功 |
| 语法关系 | 主谓结构 | 太阳：升起 |
| | 动宾结构 | 阅读：书籍 |
| | 偏正结构 | 美丽：花朵 |

### 解题技巧

1. **先横向**：确定题干词语间的关系
2. **再纵向**：对比选项，选与题干关系最一致的
3. **二级辨析**：当多个选项关系相同时，进一步比较词性、感情色彩、主被动等

## 五、科学推理（京考特色，⭐⭐⭐）

> 2026京考大纲明确包含科学推理，涉及物理、化学、生物基础知识的应用。

### 高频考点

| 学科 | 常考内容 |
|------|----------|
| 力学 | 受力分析、杠杆平衡、浮力、摩擦力 |
| 电学 | 串并联电路、欧姆定律、电功率 |
| 光学 | 反射/折射定律、凸透镜成像 |
| 化学 | 酸碱反应、氧化还原、金属活动性 |
| 生物 | 遗传规律、生态系统、人体健康 |

---

## 六、京考判断推理备考建议

1. **图形推理**（15题）：每天练习5-10题，建立"规律库"，重点突破空间重构
2. **逻辑判断**：熟记翻译规则，削弱加强找准论证结构
3. **定义判断**：严格依据定义文本，不要凭常识判断
4. **类比推理**：积累常见逻辑关系，注意二级辨析
5. **科学推理**：回顾初高中理科基础知识
