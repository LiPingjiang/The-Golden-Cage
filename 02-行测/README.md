# 第2章 行政职业能力测验（行测）

> 行测考的是知识面的广度和解决问题的速度，120分钟内完成约135道选择题。

---

## 2.1 考试概况

| 项目 | 内容 |
|------|------|
| 题型 | 全部为客观性单项选择题 |
| 时间 | 120分钟 |
| 满分 | 100分 |
| 总题量 | 约135题 |
| 无倒扣分 | 答错不扣分，全部作答 |

---

## 2.2 题型分布与分值结构

| 模块 | 题量 | 参考分值 | 占比 | 难度 |
|------|------|----------|------|------|
| 常识判断（含政治理论） | 30-35题 | ~21分 | ~21% | ⭐⭐ |
| 言语理解与表达 | 30-35题 | ~24分 | ~24% | ⭐⭐⭐ |
| 数量关系 | 15-20题 | ~12分 | ~12% | ⭐⭐⭐⭐ |
| 判断推理 | 30-35题 | ~25分 | ~25% | ⭐⭐⭐ |
| 资料分析 | 20题 | ~20分 | ~20% | ⭐⭐⭐ |

> ⚠️ 官方未公布各题具体分值，以上为根据历年考情推算的参考值。

---

## 2.3 京考行测特点（vs 国考）

| 对比维度 | 京考 | 国考 |
|----------|------|------|
| 总题量 | 135题 | 130-135题 |
| 言语理解 | 以篇章阅读为主，无选词填空 | 逻辑填空+片段阅读 |
| 判断推理 | 含科学推理模块 | 无科学推理 |
| 数量关系 | 含数字推理 | 仅数学运算 |
| 常识判断 | 侧重北京市情、时政 | 侧重国情 |
| 整体难度 | 中等偏上 | 较高 |

---

## 2.4 各模块性价比排序

根据"提分难度"和"分值占比"综合评估：

```
🥇 资料分析 — 分值高、规律性强、短期提分最快
🥈 判断推理 — 分值最高、方法性强
🥉 言语理解 — 占比大、需要长期积累
4️⃣ 数量关系 — 难度大但简单题占50-60%
5️⃣ 常识判断 — 靠日常积累，短期难突破
```

---

## 2.5 答题时间分配建议

| 模块 | 建议用时 | 策略 |
|------|----------|------|
| 常识判断 | 10分钟 | 快速作答，不纠结 |
| 言语理解 | 30分钟 | 先读问题再看文章 |
| 数量关系 | 15分钟 | 先做简单题，难题猜答案 |
| 判断推理 | 35分钟 | 图推控制时间，逻辑仔细推 |
| 资料分析 | 25分钟 | 必须全部做完，正确率目标90% |
| 涂卡检查 | 5分钟 | — |
