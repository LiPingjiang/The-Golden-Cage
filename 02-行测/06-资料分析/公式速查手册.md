# 资料分析核心公式速查手册

> 资料分析是行测中**最容易提分**的模块，掌握公式+速算技巧，正确率可达90%+。

---

## 一、增长相关（必考 ⭐⭐⭐⭐⭐）

### 1.1 基本概念

| 概念 | 公式 | 说明 |
|------|------|------|
| 现期量 | 现期量 = 基期量 ×（1 + r） | 当前时期的数据 |
| 基期量 | 基期量 = 现期量 ÷（1 + r） | 对比基础时期的数据 |
| 增长量 | 增长量 = 现期量 - 基期量 | 绝对增加了多少 |
| 增长率 | r = 增长量 ÷ 基期量 | 相对增加了多少 |

### 1.2 增长量计算

**公式变形**：
```
增长量 = 现期量 - 基期量
增长量 = 基期量 × r
增长量 = 现期量 × r ÷ (1 + r)
```

**速算技巧——n+1法则**：
- 若增长率 r ≈ 1/n，则增长量 ≈ 现期量 ÷ (n+1)
- 若减少率 r ≈ 1/n，则减少量 ≈ 现期量 ÷ (n-1)

| 增长率 | 近似分数 | 速算 |
|--------|----------|------|
| 10% | 1/10 | 现期量 ÷ 11 |
| 11.1% | 1/9 | 现期量 ÷ 10 |
| 12.5% | 1/8 | 现期量 ÷ 9 |
| 14.3% | 1/7 | 现期量 ÷ 8 |
| 16.7% | 1/6 | 现期量 ÷ 7 |
| 20% | 1/5 | 现期量 ÷ 6 |
| 25% | 1/4 | 现期量 ÷ 5 |
| 33.3% | 1/3 | 现期量 ÷ 4 |
| 50% | 1/2 | 现期量 ÷ 3 |

### 1.3 增长率比较

**方法**：
1. **直接读数**：材料直接给出增长率，直接比较
2. **增量替代**：基期量相近时，增长量大的增长率大
3. **倍数替代**：r = 现期量/基期量 - 1，比较倍数即可
4. **计算比较**：分数大小比较（交叉相乘）

### 1.4 间隔增长率

**适用场景**：求隔一年的增长率（如2024年比2022年）

**公式**：R = r₁ + r₂ + r₁ × r₂

> 例：2023年增长率10%，2024年增长率20%
> 间隔增长率 = 10% + 20% + 10%×20% = 32%

### 1.5 混合增长率

**适用场景**：整体 = 部分1 + 部分2

**规律**：
- 整体增长率介于各部分增长率之间
- 整体增长率偏向**基期量较大**的部分

### 1.6 年均增长量与年均增长率

| 类型 | 公式 |
|------|------|
| 年均增长量 | （末期量 - 初期量）÷ n |
| 年均增长率 | 末期量 = 初期量 ×（1 + x）ⁿ |

> 注意：n = 年份差（如2020-2015，n=5）

## 二、比重相关（必考 ⭐⭐⭐⭐⭐）

### 2.1 现期比重

**公式**：比重 = 部分量 ÷ 整体量

**识别关键词**："占""比重""比例""占比"

### 2.2 基期比重

**公式**：基期比重 = (A/B) × [(1+b)/(1+a)]

> 其中 A=部分现期量，B=整体现期量，a=部分增长率，b=整体增长率

### 2.3 两期比重变化

**判断升降**（定性）：
- 部分增长率 a > 整体增长率 b → 比重**上升**
- 部分增长率 a < 整体增长率 b → 比重**下降**
- 部分增长率 a = 整体增长率 b → 比重**不变**

**计算变化量**（定量）：
```
两期比重差 = (A/B) × (a-b)/(1+a)
```

**速算技巧**：比重变化量的绝对值 < |a - b|，选绝对值最小的选项！

## 三、平均数相关（高频 ⭐⭐⭐⭐）

### 3.1 现期平均数

**公式**：平均数 = 总量 ÷ 份数

### 3.2 基期平均数

**公式**：基期平均数 = (A/B) × [(1+b)/(1+a)]

> 其中 A=总量现期值，B=份数现期值，a=总量增长率，b=份数增长率

### 3.3 平均数变化判断

- 总量增长率 > 份数增长率 → 平均数**增加**
- 总量增长率 < 份数增长率 → 平均数**减少**

### 3.4 平均数增长率

**公式**：平均数增长率 = (a - b) ÷ (1 + b)

> 其中 a=总量增长率，b=份数增长率

---

## 四、倍数相关（⭐⭐⭐）

### 4.1 现期倍数

**公式**：A是B的多少倍 = A ÷ B

> 注意："多几倍" = "是几倍" - 1

### 4.2 基期倍数

**公式**：基期倍数 = (A/B) × [(1+b)/(1+a)]

## 五、速算技巧汇总

### 5.1 截位直除法

**核心**：分子分母同时保留前2-3位有效数字

| 选项差距 | 保留位数 | 示例 |
|----------|----------|------|
| 首位不同 | 保留2位 | 3864÷7952 → 39÷80 ≈ 0.49 |
| 首位相同 | 保留3位 | 3864÷7952 → 386÷795 ≈ 0.486 |

### 5.2 特征数字法（百化分）

| 百分数 | 分数 | 百分数 | 分数 |
|--------|------|--------|------|
| 10% | 1/10 | 16.7% | 1/6 |
| 11.1% | 1/9 | 20% | 1/5 |
| 12.5% | 1/8 | 25% | 1/4 |
| 14.3% | 1/7 | 33.3% | 1/3 |
| 15% | 3/20 | 50% | 1/2 |

### 5.3 有效数字法（乘法）

取两个数各自的前两位有效数字相乘：
- 第三位均为0-4：全舍
- 第三位均为5-9：全进
- 一大一小：小的四舍五入

### 5.4 错位加减法

适用于乘法中一个数接近整百整千的情况：
```
A × 98% = A × (1 - 2%) = A - A×2%
A × 103% = A × (1 + 3%) = A + A×3%
```

### 5.5 尾数法

适用于加减法精确计算，只看末一位或末两位即可选出答案。

## 六、常见陷阱与易错点

| 陷阱类型 | 说明 | 应对 |
|----------|------|------|
| 时间陷阱 | 问的是基期还是现期 | 仔细审题，圈出时间 |
| 单位陷阱 | 亿元 vs 万元，% vs ‰ | 统一单位再计算 |
| 增长率 vs 增长量 | "增长最多"≠"增长最快" | 量看绝对值，率看相对值 |
| "是"vs"多" | "是几倍"和"多几倍"差1 | 多几倍 = 是几倍 - 1 |
| 比重升降 | 比重变化用百分点不用% | 注意单位表述 |
| 同比 vs 环比 | 同比=与去年同期比，环比=与上期比 | 明确对比基准 |

## 七、做题流程（建议1分钟/题）

```
1. 读问题 → 确定考点类型（增长/比重/平均/倍数）
2. 定位材料 → 找到对应数据
3. 列式 → 套用对应公式
4. 速算 → 选择合适的速算方法
5. 验证 → 看选项差距决定精度
```
