# 2.6 资料分析

> 性价比之王：分值高、规律性强、短期提分最快，正确率目标90%+。

---

## 考查内容

资料分析主要测查对各种形式的文字、图表等资料的综合理解与分析加工能力。

---

## 题型结构

- 通常4篇材料，每篇5道题，共20题
- 材料类型：文字型、表格型、图表型、综合型
- 每题约1分，总分约20分

---

## 核心概念与公式

### 增长

| 概念 | 公式 |
|------|------|
| 增长量 | 现期量 - 基期量 |
| 增长率 | 增长量 ÷ 基期量 × 100% |
| 基期量 | 现期量 ÷ (1 + 增长率) |
| 间隔增长率 | r₁ + r₂ + r₁×r₂ |

### 比重

| 概念 | 公式 |
|------|------|
| 比重 | 部分 ÷ 整体 × 100% |
| 基期比重 | (A/B) × [(1+b)/(1+a)] |
| 比重变化 | 判断a与b的大小关系 |

### 倍数

| 概念 | 公式 |
|------|------|
| 是几倍 | A ÷ B |
| 多几倍 | (A-B) ÷ B = A÷B - 1 |
| 基期倍数 | (A/B) × [(1+b)/(1+a)] |

### 平均数

| 概念 | 公式 |
|------|------|
| 平均数 | 总量 ÷ 个数 |
| 平均数增长率 | (a-b) ÷ (1+b) |

---

## 速算技巧

### 截位直除
- 保留前2-3位有效数字
- 分母截位，分子不变或同步截位
- 适用于大部分除法运算

### 分数比较
- 直除看首位
- 交叉相乘比较大小

### 特殊值记忆
- 1/3 ≈ 33.3%
- 1/6 ≈ 16.7%
- 1/7 ≈ 14.3%
- 1/8 = 12.5%
- 1/9 ≈ 11.1%

### 增长量比较
- 大大则大（现期量大、增长率大→增长量大）
- 一大一小用公式：增长量 = 现期量 × r/(1+r)

---

## 常见陷阱

| 陷阱类型 | 说明 |
|----------|------|
| 时间陷阱 | 问的是2023年，材料给的是2024年数据 |
| 单位陷阱 | 亿元 vs 万元，% vs ‰ |
| 同比/环比 | 同比=与去年同期比，环比=与上一期比 |
| 增长率vs增长量 | 问的是率还是量 |
| "约"字 | 选最接近的，不必精确计算 |

## 备考策略

### 训练方法
1. **每天必练**：固定30分钟，4篇材料20题
2. **限时训练**：每篇材料+5题控制在6分钟内
3. **错题复盘**：分析是概念错误还是计算错误
4. **公式熟练**：核心公式必须条件反射

### 正确率目标
- 基础阶段：75%+
- 提升阶段：85%+
- 冲刺阶段：90%+（20题错2题以内）

### 考场策略
- 资料分析建议放在前面做（头脑清醒时）
- 综合判断题（以下说法正确/错误的是）可以先验证最容易判断的选项
- 遇到复杂计算先估算，看选项差距决定精度

## 典型例题精讲

### 例题1：资料分析 — 增长率计算

**材料**：某市2024年第三季度新鲜水果总产量为45.8万吨，其中苹果产量占新鲜水果总产量的比重为68%。已知苹果产量比第二季度的产量多12%。

**问题**：该市2024年第二季度苹果产量约为多少万吨？

A. 24.5　　B. 27.8　　C. 31.1　　D. 34.9

**解析**：
1. 第三季度苹果产量 = 45.8 × 68% = 31.144万吨
2. 设第二季度苹果产量为X，则 X ×（1+12%）= 31.144
3. X = 31.144 ÷ 1.12 ≈ 27.8万吨

**答案**：B

**考点**：基期量 = 现期量 ÷（1 + 增长率）

---

### 例题2：资料分析 — 比重比较

**材料**：2023年全国高技术产业主要指标：
- 规模以上企业数量：15.9万个，同比增长9.5%
- 营业收入：29.8万亿元，同比增长11.2%
- 工业增加值：10.5万亿元，同比增长12.3%
- 利润总额：2.3万亿元，同比增长14.8%
- 从业人员：436万人，同比增长5.2%

其中，计算机、通信和其他电子设备制造业：营业收入15.3万亿元，从业人员287万人。

**问题**：2023年，计算机、通信和其他电子设备制造业营业收入占全部高技术产业营业收入的比重约为：

A. 48.7%　　B. 51.3%　　C. 53.8%　　D. 56.2%

**解析**：
比重 = 部分 ÷ 整体 = 15.3 ÷ 29.8

截位直除：153 ÷ 298 ≈ 153 ÷ 300 = 0.51，精确计算 153÷298 ≈ 0.5134

**答案**：B（51.3%）

**考点**：比重计算 + 截位直除

### 例题3：资料分析 — 增长量比较

**问题**（接上题材料）：2023年以下哪个指标的增长量最大？

A. 企业数量　　B. 营业收入　　C. 利润总额　　D. 从业人员

**解析**：
增长量 = 现期量 × r/(1+r)

- 企业数量：15.9 × 9.5%/109.5% ≈ 15.9 × 0.087 ≈ 1.38万个
- 营业收入：29.8 × 11.2%/111.2% ≈ 29.8 × 0.1007 ≈ 3.0万亿元
- 利润总额：2.3 × 14.8%/114.8% ≈ 2.3 × 0.129 ≈ 0.30万亿元
- 从业人员：436 × 5.2%/105.2% ≈ 436 × 0.049 ≈ 21.5万人

注意：这道题各指标**单位不同**，不能直接比较绝对数值。题目问的是"增长量最大"，需要在同一单位体系下比较。如果题目问的是"增长量最多的是"且选项单位一致，则选营业收入（增长约3万亿元）。

**答案**：B

**考点**：增长量公式 + 注意单位陷阱

### 例题4：资料分析 — 间隔增长率

**题目**：某地区2022年GDP为5000亿元，2023年同比增长8%，2024年同比增长6%。则2024年GDP相比2022年增长了约百分之几？

A. 14%　　B. 14.48%　　C. 14.5%　　D. 15%

**解析**：
间隔增长率公式：r = r₁ + r₂ + r₁ × r₂
= 8% + 6% + 8% × 6%
= 14% + 0.48%
= 14.48%

**答案**：B

**考点**：间隔增长率 = r₁ + r₂ + r₁×r₂

### 例题5：资料分析 — 基期比重

**题目**：2024年某市第三产业增加值为21439亿元，同比增长5.2%；GDP为25755.3亿元，同比增长4.8%。则2023年该市第三产业增加值占GDP的比重约为：

A. 82.8%　　B. 83.0%　　C. 83.3%　　D. 83.7%

**解析**：
基期比重 = (A/B) × [(1+b)/(1+a)]

现期比重 = 21439/25755.3 ≈ 83.2%

基期比重 = 83.2% × (1+4.8%)/(1+5.2%) = 83.2% × 104.8%/105.2%

由于 b < a（4.8% < 5.2%），所以基期比重 < 现期比重

104.8/105.2 ≈ 0.9962

83.2% × 0.9962 ≈ 82.9% ≈ 83.0%

**答案**：B

**考点**：基期比重公式，判断比重升降（a>b则比重上升）
