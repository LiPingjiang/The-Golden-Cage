# 2.6 资料分析

> 性价比之王：分值高、规律性强、短期提分最快，正确率目标90%+。

---

## 考查内容

资料分析主要测查对各种形式的文字、图表等资料的综合理解与分析加工能力。

---

## 题型结构

- 通常4篇材料，每篇5道题，共20题
- 材料类型：文字型、表格型、图表型、综合型
- 每题约1分，总分约20分

---

## 核心概念与公式

### 增长

| 概念 | 公式 |
|------|------|
| 增长量 | 现期量 - 基期量 |
| 增长率 | 增长量 ÷ 基期量 × 100% |
| 基期量 | 现期量 ÷ (1 + 增长率) |
| 间隔增长率 | r₁ + r₂ + r₁×r₂ |

### 比重

| 概念 | 公式 |
|------|------|
| 比重 | 部分 ÷ 整体 × 100% |
| 基期比重 | (A/B) × [(1+b)/(1+a)] |
| 比重变化 | 判断a与b的大小关系 |

### 倍数

| 概念 | 公式 |
|------|------|
| 是几倍 | A ÷ B |
| 多几倍 | (A-B) ÷ B = A÷B - 1 |
| 基期倍数 | (A/B) × [(1+b)/(1+a)] |

### 平均数

| 概念 | 公式 |
|------|------|
| 平均数 | 总量 ÷ 个数 |
| 平均数增长率 | (a-b) ÷ (1+b) |

## 速算技巧

### 截位直除
- 保留前2-3位有效数字
- 分母截位，分子不变或同步截位
- 适用于大部分除法运算

### 分数比较
- 直除看首位
- 交叉相乘比较大小

### 特殊值记忆
- 1/3 ≈ 33.3%
- 1/6 ≈ 16.7%
- 1/7 ≈ 14.3%
- 1/8 = 12.5%
- 1/9 ≈ 11.1%

### 增长量比较
- 大大则大（现期量大、增长率大→增长量大）
- 一大一小用公式：增长量 = 现期量 × r/(1+r)

## 常见陷阱

| 陷阱类型 | 说明 |
|----------|------|
| 时间陷阱 | 问的是2023年，材料给的是2024年数据 |
| 单位陷阱 | 亿元 vs 万元，% vs ‰ |
| 同比/环比 | 同比=与去年同期比，环比=与上一期比 |
| 增长率vs增长量 | 问的是率还是量 |
| "约"字 | 选最接近的，不必精确计算 |

## 备考策略

### 训练方法
1. **每天必练**：固定30分钟，4篇材料20题
2. **限时训练**：每篇材料+5题控制在6分钟内
3. **错题复盘**：分析是概念错误还是计算错误
4. **公式熟练**：核心公式必须条件反射

### 正确率目标
- 基础阶段：75%+
- 提升阶段：85%+
- 冲刺阶段：90%+（20题错2题以内）

### 考场策略
- 资料分析建议放在前面做（头脑清醒时）
- 综合判断题（以下说法正确/错误的是）可以先验证最容易判断的选项
- 遇到复杂计算先估算，看选项差距决定精度
