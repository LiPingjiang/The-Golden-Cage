> 覆盖面最广的模块，靠日常积累，不建议花大块时间专攻。

## 考查内容

常识判断主要测查报考者应知应会的基本知识以及运用这些知识分析判断的基本能力。

### 考查范围

| 领域 | 高频考点 |
|------|----------|
| 政治 | 时事政治、北京市政策、重大会议 |
| 法律 | 宪法、行政法、民法典、监察法、新修订法律 |
| 经济 | 宏观经济政策、市场经济基础 |
| 科技 | 前沿科技、生活常识、物理化学生物 |
| 人文 | 中国历史、文学常识、传统文化 |
| 地理 | 中国地理、世界地理、气候 |
| 北京市情 | 历史沿革、行政区划、城市规划、文化遗产 |

## 京考常识特点

- **北京市情占比高**：每年必考北京相关时政和市情
- **科技生活化**：垃圾分类、低碳生活、食品安全等

## 北京市情知识点汇编

> 京考常识高频考点，务必掌握。

### 一、"四个中心"功能建设

| 中心 | 内涵 | 重点工作 |
|------|------|----------|
| 全国政治中心 | 服务保障中央政务功能 | 央地协同、政务环境优化、核心区控规实施 |
| 全国文化中心 | 传承历史文脉、引领文化创新 | 中轴线申遗、三条文化带建设、文化产业发展 |
| 国际交往中心 | 服务国家外交大局 | 雁栖湖、国家会议中心、第二使馆区建设 |
| 科技创新中心 | 建设国际科技创新中心 | 三城一区、中关村示范区、原始创新策源 |

### 二、城市空间结构："一核一主一副、两轴多点一区"

| 要素 | 范围 | 面积 | 功能定位 |
|------|------|------|----------|
| 一核 | 东城区+西城区 | 92.5km² | 首都功能核心区 |
| 一主 | 城六区 | 1378km² | 中心城区，疏解非首都功能 |
| 一副 | 通州 | 155km² | 城市副中心，行政办公+商务+文旅 |
| 两轴 | 中轴线+长安街 | — | 统领城市空间秩序 |
| 多点 | 顺义、大兴、亦庄、昌平、房山 | — | 承接中心城区功能 |
| 一区 | 门头沟、平谷、怀柔、密云、延庆 | — | 生态涵养区 |

### 三、北京城市副中心

- **位置**：通州区
- **定位**：国际一流的和谐宜居现代化城区
- **功能**：行政办公、商务服务、文化旅游、科技创新
- **与中心城区关系**：形成"双城"格局
- **与雄安新区关系**：北京新两翼，错位承接非首都功能
- **重大项目**：城市绿心森林公园、三大文化设施（图书馆、剧院、博物馆）、M101线

### 四、北京中轴线

- **全长**：约7.8公里
- **起止**：南起永定门，北至钟鼓楼
- **历史**：始建于13世纪（元大都），历经元明清及近现代逾7个世纪
- **申遗**：2012年启动，**2024年成功列入《世界遗产名录》**，历时12年
- **遗产点**：15处，包括故宫、天坛、天安门广场、先农坛、太庙等，全部对外开放

### 一、宪法核心知识

| 知识点 | 内容 |
|--------|------|
| 国体 | 人民民主专政的社会主义国家 |
| 政体 | 人民代表大会制度 |
| 国家结构形式 | 单一制 |
| 基本经济制度 | 公有制为主体、多种所有制经济共同发展 |
| 分配制度 | 按劳分配为主体、多种分配方式并存 |
| 公民基本权利 | 平等权、政治权利、人身自由、社会经济权利、文化教育权利 |
| 宪法修改 | 全国人大常委会或1/5以上全国人大代表提议，全国人大2/3以上多数通过 |
| 宪法解释 | 全国人大常委会 |
| 宪法监督 | 全国人大及其常委会 |

### 二、行政法（高频）

| 知识点 | 要点 |
|--------|------|
| 行政处罚种类 | 警告/通报批评、罚款、没收、暂扣/吊销许可证、限制/禁止从业、行政拘留 |
| 行政处罚管辖 | 违法行为发生地的县级以上地方人民政府有管辖权的行政机关 |
| 简易程序 | 违法事实确凿、有法定依据、对公民200元以下/对法人3000元以下罚款 |
| 行政许可设定 | 法律、行政法规、地方性法规可设定；规章可设临时性许可 |
| 行政强制种类 | 查封、扣押、冻结、限制人身自由（行政强制措施）；加处罚款、划拨存款、拍卖等（行政强制执行） |
| 行政复议期限 | 知道具体行政行为之日起60日内 |
| 行政诉讼期限 | 知道或应当知道之日起6个月内 |

## 常识判断 · 经济常识核心知识点（补充）

> 经济常识在近三年京考中稳定占2-5题，重点是宏观经济调控和常用经济指标。

### 一、宏观经济调控

**宏观调控四大目标**：促进经济增长、增加就业、稳定物价、保持国际收支平衡

**调控三大手段**：经济手段（主要）、法律手段、行政手段（辅助）

**经济手段核心**：财政政策 + 货币政策

### 二、财政政策（财政部制定）

| 类型 | 适用场景 | 具体措施 |
|------|----------|----------|
| 积极（扩张性）财政政策 | 经济衰退/内需不足 | 增加政府支出、增加转移支付、减少税收、发行国债 |
| 消极（紧缩性）财政政策 | 经济过热/通货膨胀 | 减少政府支出、增加税收 |

**财政收入构成**：税、利、债、费

**财政支出构成**：政府购买 + 转移支付

### 三、货币政策（中国人民银行制定，不是"中国银行"）

| 工具 | 经济萧条（宽松） | 经济过热（紧缩） |
|------|-----------------|-----------------|
| 法定存款准备金率 | 降低 | 提高 |
| 再贴现率 | 降低 | 提高 |
| 公开市场业务 | 买入债券 | 卖出债券 |
| 基准利率 | 降低 | 提高 |

**易混淆点**：
- 制定货币政策的主体是**中国人民银行**（不是"中国银行"，中国银行是商业银行）
- 财政政策关键词：税收、财政收支、国债
- 货币政策关键词：利率、存款准备金率、汇率、公开市场业务

### 四、常考经济指标

| 指标 | 含义 | 记忆要点 |
|------|------|----------|
| **GDP**（国内生产总值） | 一定时期内一个国家/地区生产的全部最终产品和服务价值 | 支出法：GDP = 消费C + 投资I + 政府购买G + 净出口(X-M) |
| **GNP**（国民生产总值） | 一国**居民**在一年内生产的产品和劳务市场价值总额 | 按国籍统计，GDP按地域统计 |
| **CPI**（居民消费价格指数） | 反映居民购买消费品价格水平变动 | 涨幅>3%为通货膨胀，>5%为严重通货膨胀 |
| **PPI**（生产者价格指数） | 反映生产环节价格变动 | 领先CPI变化 |
| **基尼系数** | 衡量收入分配公平程度 | 0-1之间，越小越公平；**0.4为收入差距警戒线** |
| **恩格尔系数** | 食品支出占消费支出比重 | 越低生活水平越高（国际通用生活水平指标） |

### 五、通货膨胀与通货紧缩

| 现象 | 定义 | 应对政策 |
|------|------|----------|
| 通货膨胀 | 物价持续上涨，货币购买力下降 | 紧缩性财政+货币政策 |
| 通货紧缩 | 物价持续下跌，货币供应量、物价、经济增长率同时下降，失业率上升 | 扩张性财政+货币政策 |

### 六、经济组织与经济名词（近年新增考点）

| 名词 | 含义 |
|------|------|
| 机会成本 | 选择某方案而放弃其他方案中最优方案的价值 |
| 沉没成本 | 已经发生不可回收的成本，不应影响未来决策 |
| 边际效应 | 每增加一个单位投入所带来的额外产出变化 |
| 规模经济 | 随生产规模扩大，单位成本下降 |
| 帕累托最优 | 资源配置达到无法在不损害他人利益前提下改善任何人处境的状态 |

### 七、答题技巧

- 遇到"两个相反选项"（如提高/降低利率），答案通常在其中选一个
- "太绝对"的表述（如"唯一""必然""绝对"）要提高警惕，往往是错误项
- 分清主体：财政政策→财政部；货币政策→中国人民银行；产业政策→发改委/工信部
