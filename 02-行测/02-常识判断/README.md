# 2.2 常识判断

> 覆盖面最广的模块，靠日常积累，不建议花大块时间专攻。

---

## 考查内容

常识判断主要测查报考者应知应会的基本知识以及运用这些知识分析判断的基本能力。

### 考查范围

| 领域 | 高频考点 |
|------|----------|
| 政治 | 时事政治、北京市政策、重大会议 |
| 法律 | 宪法、行政法、民法典、监察法、新修订法律 |
| 经济 | 宏观经济政策、市场经济基础 |
| 科技 | 前沿科技、生活常识、物理化学生物 |
| 人文 | 中国历史、文学常识、传统文化 |
| 地理 | 中国地理、世界地理、气候 |
| 北京市情 | 历史沿革、行政区划、城市规划、文化遗产 |

---

## 京考常识特点

- **北京市情占比高**：每年必考北京相关时政和市情
- **科技生活化**：垃圾分类、低碳生活、食品安全等
- **法律侧重新法**：关注当年新修订或新颁布的法律
- **题量变化**：2024年起从35题调整为30题，不再考多选

---

## 备考策略

### 推荐做法
- ✅ 利用碎片时间学习，每天15-20分钟
- ✅ 关注北京本地新闻和政策
- ✅ 重点掌握近1-2年新修订法律
- ✅ 做近3年京考真题常识部分

### 不推荐做法
- ❌ 花大块时间系统背诵
- ❌ 追求面面俱到
- ❌ 在冷门知识点上深挖

---

## 高频考点清单

### 法律必备
- 宪法基本制度
- 行政处罚法、行政许可法
- 民法典（物权、合同、婚姻家庭）
- 刑法修正案
- 民营经济促进法（2025年新法）

### 北京市情必备
- "四个中心"：政治中心、文化中心、国际交往中心、科技创新中心
- 中轴线申遗
- 城市副中心建设
- 接诉即办改革
