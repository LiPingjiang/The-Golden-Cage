# 2.2 常识判断

> 覆盖面最广的模块，靠日常积累，不建议花大块时间专攻。

---

## 考查内容

常识判断主要测查报考者应知应会的基本知识以及运用这些知识分析判断的基本能力。

### 考查范围

| 领域 | 高频考点 |
|------|----------|
| 政治 | 时事政治、北京市政策、重大会议 |
| 法律 | 宪法、行政法、民法典、监察法、新修订法律 |
| 经济 | 宏观经济政策、市场经济基础 |
| 科技 | 前沿科技、生活常识、物理化学生物 |
| 人文 | 中国历史、文学常识、传统文化 |
| 地理 | 中国地理、世界地理、气候 |
| 北京市情 | 历史沿革、行政区划、城市规划、文化遗产 |

---

## 京考常识特点

- **北京市情占比高**：每年必考北京相关时政和市情
- **科技生活化**：垃圾分类、低碳生活、食品安全等
- **法律侧重新法**：关注当年新修订或新颁布的法律
- **题量变化**：2024年起从35题调整为30题，不再考多选

---

## 备考策略

### 推荐做法
- ✅ 利用碎片时间学习，每天15-20分钟
- ✅ 关注北京本地新闻和政策
- ✅ 重点掌握近1-2年新修订法律
- ✅ 做近3年京考真题常识部分

### 不推荐做法
- ❌ 花大块时间系统背诵
- ❌ 追求面面俱到
- ❌ 在冷门知识点上深挖

## 北京市情知识点汇编

> 京考常识高频考点，务必掌握。

### 一、"四个中心"功能建设

| 中心 | 内涵 | 重点工作 |
|------|------|----------|
| 全国政治中心 | 服务保障中央政务功能 | 央地协同、政务环境优化、核心区控规实施 |
| 全国文化中心 | 传承历史文脉、引领文化创新 | 中轴线申遗、三条文化带建设、文化产业发展 |
| 国际交往中心 | 服务国家外交大局 | 雁栖湖、国家会议中心、第二使馆区建设 |
| 科技创新中心 | 建设国际科技创新中心 | 三城一区、中关村示范区、原始创新策源 |

### 二、城市空间结构："一核一主一副、两轴多点一区"

| 要素 | 范围 | 面积 | 功能定位 |
|------|------|------|----------|
| 一核 | 东城区+西城区 | 92.5km² | 首都功能核心区 |
| 一主 | 城六区 | 1378km² | 中心城区，疏解非首都功能 |
| 一副 | 通州 | 155km² | 城市副中心，行政办公+商务+文旅 |
| 两轴 | 中轴线+长安街 | — | 统领城市空间秩序 |
| 多点 | 顺义、大兴、亦庄、昌平、房山 | — | 承接中心城区功能 |
| 一区 | 门头沟、平谷、怀柔、密云、延庆 | — | 生态涵养区 |

### 三、北京城市副中心

- **位置**：通州区
- **定位**：国际一流的和谐宜居现代化城区
- **功能**：行政办公、商务服务、文化旅游、科技创新
- **与中心城区关系**：形成"双城"格局
- **与雄安新区关系**：北京新两翼，错位承接非首都功能
- **重大项目**：城市绿心森林公园、三大文化设施（图书馆、剧院、博物馆）、M101线

### 四、北京中轴线

- **全长**：约7.8公里
- **起止**：南起永定门，北至钟鼓楼
- **历史**：始建于13世纪（元大都），历经元明清及近现代逾7个世纪
- **申遗**：2012年启动，**2024年成功列入《世界遗产名录》**，历时12年
- **遗产点**：15处，包括故宫、天坛、天安门广场、先农坛、太庙等，全部对外开放

### 五、接诉即办

| 要素 | 内容 |
|------|------|
| 平台 | 12345市民服务热线 |
| 启动时间 | 2019年 |
| 立法 | 《北京市接诉即办工作条例》2021年9月24日通过 |
| 核心理念 | 民有所呼、我有所应 |
| 办理时限 | 一般7个自然日，复杂事项经批准延长至15个自然日 |
| 制度特色 | 首接负责制、"每月一题"机制、未诉先办 |
| 评价维度 | 响应率、解决率、满意率 |
| 改革方向 | 从"有一办一"到"举一反三"，从"接诉即办"到"未诉先办" |

### 六、"三城一区"

| 名称 | 位置 | 定位 |
|------|------|------|
| 中关村科学城 | 海淀区 | 原始创新策源地 |
| 怀柔科学城 | 怀柔区 | 大科学装置集群，"科学+城"融合 |
| 未来科学城 | 昌平区 | 央企研发、创新创业 |
| 北京经济技术开发区（亦庄） | 大兴区 | 高精尖产业主阵地 |

### 七、"两区"建设

- **国家服务业扩大开放综合示范区**
- **中国（北京）自由贸易试验区**
  - 科技创新片区（海淀）
  - 国际商务服务片区（朝阳、通州）
  - 高端产业片区（大兴机场临空经济区）

### 八、京津冀协同发展

| 要素 | 内容 |
|------|------|
| 战略提出 | 2014年习近平总书记提出 |
| 纲要批复 | 2015年《京津冀协同发展规划纲要》 |
| 核心任务 | 疏解北京非首都功能 |
| 三大重点 | 交通一体化、生态环保、产业升级转移 |
| 两翼 | 北京城市副中心 + 雄安新区 |

### 九、北京生态环境（高频数据）

| 指标 | 2024年数据 |
|------|-----------|
| PM2.5年均浓度 | 28微克/立方米 |

| 劣Ⅴ类水体 | 动态清零（2023年完成） |

### 十、2026年政府工作报告关键词

- "五子"联动服务和融入新发展格局

- 经济向新向优发展
- 人民城市人民建、人民城市为人民

## 常识判断 · 法律知识点汇编

### 一、宪法核心知识

| 知识点 | 内容 |
|--------|------|
| 国体 | 人民民主专政的社会主义国家 |
| 政体 | 人民代表大会制度 |
| 国家结构形式 | 单一制 |
| 基本经济制度 | 公有制为主体、多种所有制经济共同发展 |
| 分配制度 | 按劳分配为主体、多种分配方式并存 |
| 公民基本权利 | 平等权、政治权利、人身自由、社会经济权利、文化教育权利 |
| 宪法修改 | 全国人大常委会或1/5以上全国人大代表提议，全国人大2/3以上多数通过 |
| 宪法解释 | 全国人大常委会 |
| 宪法监督 | 全国人大及其常委会 |

### 二、行政法（高频）

| 知识点 | 要点 |
|--------|------|
| 行政处罚种类 | 警告/通报批评、罚款、没收、暂扣/吊销许可证、限制/禁止从业、行政拘留 |
| 行政处罚管辖 | 违法行为发生地的县级以上地方人民政府有管辖权的行政机关 |
| 简易程序 | 违法事实确凿、有法定依据、对公民200元以下/对法人3000元以下罚款 |
| 行政许可设定 | 法律、行政法规、地方性法规可设定；规章可设临时性许可 |
| 行政强制种类 | 查封、扣押、冻结、限制人身自由（行政强制措施）；加处罚款、划拨存款、拍卖等（行政强制执行） |
| 行政复议期限 | 知道具体行政行为之日起60日内 |
| 行政诉讼期限 | 知道或应当知道之日起6个月内 |

### 三、民法典核心

| 编 | 高频考点 |
|----|----------|
| 总则编 | 民事行为能力（18岁完全/8岁限制/不满8岁无）、诉讼时效3年 |
| 物权编 | 物权变动（不动产登记/动产交付）、建筑物区分所有权 |
| 合同编 | 合同成立（要约+承诺）、违约责任、格式条款 |
| 人格权编 | 隐私权、个人信息保护 |
| 婚姻家庭编 | 结婚年龄（男22女20）、离婚冷静期30天、夫妻共同财产 |
| 继承编 | 法定继承顺序（第一顺序：配偶、子女、父母）、遗嘱形式 |
| 侵权责任编 | 过错责任、无过错责任、高空抛物责任 |

### 四、刑法基础

| 知识点 | 内容 |
|--------|------|
| 刑事责任年龄 | 16岁完全负责；12-14岁经最高检核准追诉故意杀人等；14-16岁负责8种严重犯罪 |
| 正当防卫 | 对正在进行的不法侵害采取制止行为，未明显超过必要限度 |
| 紧急避险 | 为避免正在发生的危险，不得已损害另一较小合法权益 |
| 犯罪形态 | 犯罪预备、犯罪未遂、犯罪中止、犯罪既遂 |
| 主刑 | 管制、拘役、有期徒刑、无期徒刑、死刑 |
| 附加刑 | 罚金、剥夺政治权利、没收财产、驱逐出境 |

### 五、近年新法/修订法（重点关注）

| 法律 | 时间 | 要点 |
|------|------|------|
| 民营经济促进法 | 2025年 | 首部促进民营经济发展的基础性法律 |
| 反电信网络诈骗法 | 2022年 | 电信、金融、互联网行业治理 |
| 反食品浪费法 | 2021年 | 餐饮服务经营者不得诱导超量点餐 |
| 个人信息保护法 | 2021年 | 知情同意原则、最小必要原则 |
| 数据安全法 | 2021年 | 数据分类分级保护 |
| 监察法实施条例 | 2021年 | 监察机关职责权限程序 |
| 北京市接诉即办工作条例 | 2021年 | 首接负责制、7日办理时限 |
| 北京城市更新条例 | 2023年 | 留改拆结合、加装电梯规则 |
