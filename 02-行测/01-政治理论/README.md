# 2.1 政治理论

## 一、习近平新时代中国特色社会主义思想

### 1.1 "十个明确"（核心内容，逐条记忆）

1. 明确中国特色社会主义**最本质的特征**是中国共产党领导，中国特色社会主义制度的**最大优势**也是中国共产党领导，中国共产党是**最高政治领导力量**
2. 明确中国特色社会主义**总任务**是实现社会主义现代化和中华民族伟大复兴，在全面建成小康社会基础上，分两步走全面建成社会主义现代化强国
3. 明确新时代我国社会主要矛盾是**人民日益增长的美好生活需要**和**不平衡不充分的发展**之间的矛盾，必须坚持以人民为中心的发展思想
4. 明确中国特色社会主义事业总体布局是**"五位一体"**（经济、政治、文化、社会、生态文明建设），战略布局是**"四个全面"**（全面建成社会主义现代化强国、全面深化改革、全面推进依法治国、全面加强党的建设）
5. 明确全面深化改革总目标是**完善和发展中国特色社会主义制度、推进国家治理体系和治理能力现代化**
6. 明确全面推进依法治国总目标是**建设中国特色社会主义法治体系、建设社会主义法治国家**
7. 明确党在新时代的强军目标是**建设一支听党指挥、能打胜仗、作风优良的人民军队**，把人民军队建设成为世界一流军队
8. 明确中国特色大国外交要**推动构建新型国际关系，推动构建人类命运共同体**
9. 明确中国特色社会主义**最本质的特征**是中国共产党领导，中华民族伟大复兴关键在党
10. 明确全面推进**党的建设新的伟大工程**，总要求是**坚持和加强党的全面领导，坚持党要管党、全面从严治党**

> **速记口诀**：领导地位→总任务→主要矛盾→总体布局→改革目标→法治目标→强军目标→外交理念→复兴关键→党建工程

### 1.2 "十四个坚持"（基本方略）

1. 坚持党对一切工作的领导
2. 坚持以人民为中心
3. 坚持全面深化改革

5. 坚持人民当家作主
6. 坚持全面依法治国
7. 坚持社会主义核心价值体系
8. 坚持在发展中保障和改善民生
9. 坚持人与自然和谐共生
10. 坚持总体国家安全观
11. 坚持党对人民军队的绝对领导
12. 坚持"一国两制"和推进祖国统一
13. 坚持推动构建人类命运共同体

## 三、重大政策与战略部署

### 3.1 高质量发展

**定义**：高质量发展是全面建成社会主义现代化强国的**首要任务**。

**要点**：
- 完整、准确、全面贯彻新发展理念（创新、协调、绿色、开放、共享）
- 加快构建新发展格局，推动高质量发展
- 建设现代化产业体系

### 3.2 新发展格局

**核心概念**：以国内大循环为主体、国内国际双循环相互促进的新发展格局

**关键词**：
- 畅通国民经济循环
- 建设全国统一大市场
- 扩大内需战略与深化供给侧结构性改革有机结合

### 3.3 共同富裕

**基本原则**：坚持"两个毫不动摇"（毫不动摇巩固和发展公有制经济，毫不动摇支持和引导非公有制经济发展）

**实现路径**：
- 做大蛋糕（发展） + 分好蛋糕（分配）
- 初次分配、再分配、**三次分配**（慈善捐赠等）协调配套的基础性制度安排
- 防止两极分化，规范财富积累机制

### 3.4 全过程人民民主

**定义**：民主选举、民主协商、民主决策、民主管理、民主监督全过程覆盖，涵盖国家政治生活和社会生活各方面。

**制度载体**：
- 人民代表大会制度（根本政治制度）
- 中国共产党领导的多党合作和政治协商制度
- 基层民主协商（如"接诉即办"就是基层民主实践的一种）

## 四、北京市重要政策

### 4.1 首都功能定位——"四个中心"（必背，最高频）

| 定位 | 关键内涵 |
|------|----------|
| **全国政治中心** | 服务保障党中央治国理政、中央政务活动 |
| **全国文化中心** | 历史文化名城保护、全国文化中心建设、"博物馆之城" |
| **国际交往中心** | 承载国际组织总部、重大外交活动、"一带一路"国际合作高峰论坛 |
| **国际科技创新中心** | 中关村科学城、怀柔科学城、未来科学城、北京经济技术开发区"三城一区" |

> **易错点**：北京**不是**"经济中心"或"金融中心"（上海是国际经济中心和国际金融中心）；北京定位中没有"国际贸易中心"。

### 4.2 京津冀协同发展

**上升为国家战略时间**：2014年（习近平总书记在北京考察时提出）

**核心**：有序疏解北京**非首都功能**（"牛鼻子"）

**《京津冀协同发展规划纲要》**：2015年4月中央政治局会议审议通过

**三大率先突破领域**：
1. 交通一体化
2. 生态环境保护
3. 产业升级转移

**北京"一核两翼"格局**：
- "一核"：北京
- "两翼"：北京城市副中心（通州）+ 河北雄安新区

**《现代化首都都市圈空间协同规划（2023-2035年）》**：2026年初由党中央、国务院正式批复，是全国首个由党中央、国务院批复的都市圈规划。

### 4.3 北京城市总体规划（2016年-2035年）

**城市空间结构**："一核一主一副、两轴多点一区"

| 结构 | 具体内容 |
|------|----------|
| 一核 | 首都功能核心区（东城区、西城区），约92.5平方公里 |
| 一主 | 中心城区（城六区：东城、西城、朝阳、海淀、丰台、石景山） |
| 一副 | 北京城市副中心（通州区），约155平方公里 |
| 两轴 | 中轴线及其延长线、长安街及其延长线 |
| 多点 | 顺义、大兴、亦庄、昌平、房山**5个新城**（注意不是城六区） |
| 一区 | 生态涵养区（门头沟、平谷、怀柔、密云、延庆） |

**规划总目标**：建设**国际一流的和谐宜居之都**

**人口/建设用地"双控"**：常住人口规模控制在2300万人以内；城乡建设用地规模全市减量发展

### 4.4 "两区"建设

**"两区"全称**：
1. **国家服务业扩大开放综合示范区**
2. **中国（北京）自由贸易试验区**

**四大制度型开放关键词**（2026年工作要点）：
1. 制度型开放（从"首创首试"到"规则引领"）
2. 双轮驱动
3. 三区联动
4. "引进来"与"走出去"并举

**自贸试验区"四大便利"**：
- 贸易自由便利
- 投资自由便利
- 资金跨境流动便利
- 人才从业便利

## 典型例题精讲

### 例题1：北京市情（常识判断/政治理论）

**题目**：2024年北京市政府工作报告指出，要深入实施"四个中心"功能建设。下列关于"四个中心"的表述，正确的是：

A. 全国政治中心、经济中心、文化中心、科技创新中心  
B. 全国政治中心、文化中心、国际交往中心、科技创新中心  
C. 全国政治中心、文化中心、国际金融中心、科技创新中心  
D. 全国政治中心、文化中心、国际交往中心、国际贸易中心

**答案**：B

**解析**：北京城市战略定位为全国**政治中心、文化中心、国际交往中心、科技创新中心**。这是2014年习近平总书记视察北京时明确提出的，写入《北京城市总体规划（2016年-2035年）》。注意北京不是"经济中心"也不是"金融中心"（上海定位为国际金融中心）。

### 例题2：接诉即办

**题目**：下列关于《北京市接诉即办工作条例》的表述，正确的是：

A. 诉求办理时限一般为15个工作日，复杂事项可延长至30个工作日  
B. 对市民诉求实行"首接负责制"，首接单位需全程跟踪办理  
C. 未诉先办事项由街道（乡镇）单独确定，无需征求市民意见  
D. 诉求办理结果评价仅以市民满意度为唯一标准

**答案**：B

**解析**：
- A错误：一般事项办理时限为**7个自然日**，复杂事项经批准可延长至**15个自然日**
- B正确：条例规定"首接负责"，首接单位需全程跟踪，协调其他单位办理
- C错误：未诉先办事项需通过大数据分析、民意调查等方式征集市民需求
- D错误：评价标准包括响应速度、办理质量、市民满意度等多维度

### 例题3：城市空间结构

**题目**：《北京城市总体规划（2016年-2035年）》提出"一核一主一副、两轴多点一区"的城市空间结构。下列对应关系错误的是：

A. "一核"——首都功能核心区（东城区、西城区）  
B. "一副"——北京城市副中心（通州区）  
C. "两轴"——中轴线及其延长线、长安街及其延长线  
D. "多点"——朝阳、海淀、丰台、石景山、大兴5个新城

**答案**：D

**解析**：
- "一核"：首都功能核心区，约92.5平方公里（东城、西城）✓
- "一主"：中心城区，即城六区（东城、西城、朝阳、海淀、丰台、石景山）
- "一副"：北京城市副中心，原通州新城规划建设区，约155平方公里 ✓
- "两轴"：中轴线及其延长线、长安街及其延长线 ✓
- "多点"：平原地区的**顺义、大兴、亦庄、昌平、房山**5个新城（不是朝阳等城六区）
- "一区"：生态涵养区（门头沟、平谷、怀柔、密云、延庆）

### 例题4：习近平新时代中国特色社会主义思想

**题目**：党的二十大报告指出，从现在起，中国共产党的中心任务就是团结带领全国各族人民全面建成社会主义现代化强国、实现第二个百年奋斗目标，以中国式现代化全面推进中华民族伟大复兴。关于中国式现代化，下列表述错误的是：

A. 是人口规模巨大的现代化  
B. 是全体人民共同富裕的现代化  
C. 是物质文明和精神文明相协调的现代化  
D. 是以经济建设为唯一中心的现代化

**答案**：D

**解析**：中国式现代化的五个特征是：
1. 人口规模巨大的现代化
2. 全体人民共同富裕的现代化
3. 物质文明和精神文明相协调的现代化
4. 人与自然和谐共生的现代化
5. 走和平发展道路的现代化

D选项"以经济建设为唯一中心"表述错误，中国式现代化强调的是全面协调发展，不是单一维度。

A. 北京中轴线始建于13世纪，历经元、明、清及近现代逾7个世纪  
B. 北京中轴线全长约7.8公里，南起永定门，北至钟鼓楼  
C. 中轴线上的遗产点包括故宫、天坛、天安门广场等  
D. 北京中轴线申遗历时8年，于2024年成功入选

**答案**：D

**解析**：北京中轴线申遗**历时12年**（自2012年正式启动申遗工作），于2024年成功列入《世界遗产名录》，而非8年。其余选项均正确。

### 例题6：二十届三中全会（新增）

**题目**：党的二十届三中全会通过的《中共中央关于进一步全面深化改革、推进中国式现代化的决定》，明确进一步全面深化改革的总目标是到哪一年全面建成高水平社会主义市场经济体制？

A. 2027年　　B. 2029年　　C. 2035年　　D. 2049年

**答案**：C

**解析**：《决定》明确，到**2035年**，全面建成高水平社会主义市场经济体制，中国特色社会主义制度更加完善，基本实现国家治理体系和治理能力现代化，基本实现社会主义现代化。2027年是建党106周年，2029年是建国80周年，2049年是建国100周年，均非该目标年份。

### 例题7：中央城市工作会议

**题目**：2025年中央城市工作会议提出，要建设创新、宜居、美丽、韧性、文明、智慧的现代化人民城市。下列不属于这"六化"表述的是：

A. 韧性　　B. 智慧　　C. 数字　　D. 文明

**答案**：C

**解析**：中央城市工作会议明确的现代化人民城市六个特征为：**创新、宜居、美丽、韧性、文明、智慧**，不包括"数字"这一表述（数字化是智慧城市建设的手段，但官方原文用词是"智慧"而非"数字"）。
