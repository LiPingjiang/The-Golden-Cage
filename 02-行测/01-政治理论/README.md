# 2.1 政治理论

> 2025年京考新增独立模块，紧跟国考改革方向。

---

## 考查内容

政治理论是2025年京考行测新增的独立考查模块，主要测查报考者对党的创新理论的理解和运用能力。

### 核心考点

1. **习近平新时代中国特色社会主义思想**
   - 核心要义、精神实质、丰富内涵
   - "十个明确""十四个坚持""十三个方面成就"

2. **党的重要会议精神**
   - 二十大报告核心内容
   - 二十届三中全会重要决定
   - 历次中央全会公报要点

3. **重大政策与战略部署**
   - 高质量发展
   - 中国式现代化
   - 新发展格局
   - 共同富裕
   - 全过程人民民主

4. **北京市重要政策**
   - 首都功能定位（"四个中心"）
   - 京津冀协同发展
   - 北京城市总体规划
   - "两区"建设（国家服务业扩大开放综合示范区、自由贸易试验区）

---

## 备考建议

| 方法 | 说明 |
|------|------|
| 晨读原文 | 每天30分钟，读原著、学原文、悟原理 |
| 思维导图 | 梳理核心概念之间的逻辑关系 |
| 关键词区分 | 注意易混表述的细微差别 |
| 时政积累 | 关注人民日报、新华社、北京日报 |

---

## 答题技巧

- 优先选择官方标准表述
- 不纠结，控制在8-10分钟内完成
- 不确定的题目选最"官方"的选项
