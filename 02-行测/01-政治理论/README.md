# 2.1 政治理论

> 2025年京考新增独立模块，紧跟国考改革方向。

---

## 考查内容

政治理论是2025年京考行测新增的独立考查模块，主要测查报考者对党的创新理论的理解和运用能力。

### 核心考点

1. **习近平新时代中国特色社会主义思想**
   - 核心要义、精神实质、丰富内涵
   - "十个明确""十四个坚持""十三个方面成就"

2. **党的重要会议精神**
   - 二十大报告核心内容
   - 二十届三中全会重要决定
   - 历次中央全会公报要点

3. **重大政策与战略部署**
   - 高质量发展
   - 中国式现代化
   - 新发展格局
   - 共同富裕
   - 全过程人民民主

4. **北京市重要政策**
   - 首都功能定位（"四个中心"）
   - 京津冀协同发展
   - 北京城市总体规划
   - "两区"建设（国家服务业扩大开放综合示范区、自由贸易试验区）

## 典型例题精讲

### 例题1：北京市情（常识判断/政治理论）

**题目**：2024年北京市政府工作报告指出，要深入实施"四个中心"功能建设。下列关于"四个中心"的表述，正确的是：

A. 全国政治中心、经济中心、文化中心、科技创新中心  
B. 全国政治中心、文化中心、国际交往中心、科技创新中心  
C. 全国政治中心、文化中心、国际金融中心、科技创新中心  
D. 全国政治中心、文化中心、国际交往中心、国际贸易中心

**答案**：B

**解析**：北京城市战略定位为全国**政治中心、文化中心、国际交往中心、科技创新中心**。这是2014年习近平总书记视察北京时明确提出的，写入《北京城市总体规划（2016年-2035年）》。注意北京不是"经济中心"也不是"金融中心"（上海定位为国际金融中心）。

### 例题2：接诉即办

**题目**：下列关于《北京市接诉即办工作条例》的表述，正确的是：

A. 诉求办理时限一般为15个工作日，复杂事项可延长至30个工作日  
B. 对市民诉求实行"首接负责制"，首接单位需全程跟踪办理  
C. 未诉先办事项由街道（乡镇）单独确定，无需征求市民意见  
D. 诉求办理结果评价仅以市民满意度为唯一标准

**答案**：B

**解析**：
- A错误：一般事项办理时限为**7个自然日**，复杂事项经批准可延长至**15个自然日**
- B正确：条例规定"首接负责"，首接单位需全程跟踪，协调其他单位办理
- C错误：未诉先办事项需通过大数据分析、民意调查等方式征集市民需求
- D错误：评价标准包括响应速度、办理质量、市民满意度等多维度

---

### 例题3：城市空间结构

**题目**：《北京城市总体规划（2016年-2035年）》提出"一核一主一副、两轴多点一区"的城市空间结构。下列对应关系错误的是：

A. "一核"——首都功能核心区（东城区、西城区）  
B. "一副"——北京城市副中心（通州区）  
C. "两轴"——中轴线及其延长线、长安街及其延长线  
D. "多点"——朝阳、海淀、丰台、石景山、大兴5个新城

**答案**：D

**解析**：
- "一核"：首都功能核心区，约92.5平方公里（东城、西城）✓
- "一主"：中心城区，即城六区（东城、西城、朝阳、海淀、丰台、石景山）
- "一副"：北京城市副中心，原通州新城规划建设区，约155平方公里 ✓
- "两轴"：中轴线及其延长线、长安街及其延长线 ✓
- "多点"：平原地区的**顺义、大兴、亦庄、昌平、房山**5个新城（不是朝阳等城六区）
- "一区"：生态涵养区（门头沟、平谷、怀柔、密云、延庆）

### 例题4：习近平新时代中国特色社会主义思想

**题目**：党的二十大报告指出，从现在起，中国共产党的中心任务就是团结带领全国各族人民全面建成社会主义现代化强国、实现第二个百年奋斗目标，以中国式现代化全面推进中华民族伟大复兴。关于中国式现代化，下列表述错误的是：

A. 是人口规模巨大的现代化  
B. 是全体人民共同富裕的现代化  
C. 是物质文明和精神文明相协调的现代化  
D. 是以经济建设为唯一中心的现代化

**答案**：D

**解析**：中国式现代化的五个特征是：
1. 人口规模巨大的现代化
2. 全体人民共同富裕的现代化
3. 物质文明和精神文明相协调的现代化
4. 人与自然和谐共生的现代化
5. 走和平发展道路的现代化

D选项"以经济建设为唯一中心"表述错误，中国式现代化强调的是全面协调发展，不是单一维度。

### 例题5：北京中轴线

**题目**：2024年，北京中轴线成功列入《世界遗产名录》。下列关于北京中轴线的表述，不正确的是：

A. 北京中轴线始建于13世纪，历经元、明、清及近现代逾7个世纪  
B. 北京中轴线全长约7.8公里，南起永定门，北至钟鼓楼  
C. 中轴线上的遗产点包括故宫、天坛、天安门广场等  
D. 北京中轴线申遗历时8年，于2024年成功入选

**答案**：D

**解析**：北京中轴线申遗**历时12年**（自2012年正式启动申遗工作），于2024年成功列入《世界遗产名录》，而非8年。其余选项均正确。
