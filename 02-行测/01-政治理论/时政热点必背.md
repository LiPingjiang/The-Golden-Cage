# 2024-2025 时政热点必背（公务员考试重点）

> 时政是常识判断和申论的核心考点，以下整理近两年高频考点。

---

## 一、新质生产力（最高频 ⭐⭐⭐⭐⭐）

### 1.1 概念提出与发展

| 时间 | 事件 |
|------|------|
| 2023年9月 | 习近平总书记在黑龙江考察时**首次提出**"新质生产力" |
| 2023年12月 | 中央经济工作会议将"发展新质生产力"列入2024年重点工作 |
| 2024年1月 | 中央政治局第十一次集体学习，系统阐述新质生产力理论 |
| 2024年3月 | 全国两会，强调"因地制宜发展新质生产力" |
| 2025年3月 | 全国两会，指出"科技创新和产业创新，是发展新质生产力的基本路径" |

### 1.2 核心定义（规范表述）

新质生产力是**创新起主导作用**，摆脱传统经济增长方式、生产力发展路径，具有**高科技、高效能、高质量**特征，符合新发展理念的先进生产力质态。

**三个"催生"**：
- 技术革命性突破
- 生产要素创新性配置
- 产业深度转型升级

**基本内涵**：劳动者、劳动资料、劳动对象及其优化组合的跃升

**核心标志**：全要素生产率大幅提升

**特点**：创新　**关键**：质优　**本质**：先进生产力

### 1.3 发展路径（三个要点）

1. **打造新型劳动者队伍**：战略人才 + 应用型人才
2. **用好新型生产工具**：掌握关键核心技术，赋能新兴产业
3. **塑造适应新质生产力的生产关系**：打通堵点卡点，让要素顺畅流动

### 1.4 产业层面

- 巩固**战略性新兴产业**
- 提前布局**未来产业**
- 改造提升**传统产业**

## 二、北京市重大政策（京考必考 ⭐⭐⭐⭐⭐）

### 2.1 北京2025年高质量发展要点

| 领域 | 关键数据/政策 |
|------|--------------|
| 新质生产力 | 两年出台70余项细分产业支持政策 |
| 政府投资基金 | 设立8支总规模达千亿元的政府投资基金 |
| 新能源汽车 | 2024年产量达29.4万辆 |
| 集成电路 | 加快建设国家级集成电路产业基地 |
| 人工智能 | 出台通用人工智能、人形机器人等30余项政策（2024年） |
| 2025年新增 | 医疗器械、新材料等40项细分产业支持政策 |

### 2.2 北京"四个中心"建设

| 定位 | 内涵 |
|------|------|
| 全国政治中心 | 服务保障中央政务功能 |
| 全国文化中心 | 历史文化名城保护、公共文化服务 |
| 国际交往中心 | 国际组织、外交活动承载 |
| 国际科技创新中心 | 中关村、三城一区、科技成果转化 |

### 2.3 北京城市副中心

- 2023年11月发布《关于推动城市副中心高质量发展的若干措施》
- 行政办公、商务服务、文化旅游三大功能
- 绿色发展示范区、协同发展桥头堡

### 2.4 北京中轴线申遗（2024年成功）

**15处遗产点**：永定门、先农坛、天坛、正阳门及箭楼、毛主席纪念堂、人民英雄纪念碑、人民大会堂、国家博物馆、天安门、社稷坛、太庙、故宫、景山、万宁桥、鼓楼及钟楼

### 2.5 接诉即办

- 《北京市接诉即办工作条例》2021年施行
- 12345市民服务热线
- 首接负责制、7日办理时限
- "未诉先办"——主动治理

### 2.6 一刻钟便民生活圈

- 步行15分钟范围内满足居民日常生活基本消费和品质消费
- 涵盖：便利店、菜市场、早餐店、家政、维修等

## 四、重要会议与文件

### 4.1 党的二十届三中全会（2024年7月）

**主题**：进一步全面深化改革、推进中国式现代化

**核心要点**：
- 构建高水平社会主义市场经济体制
- 健全推动经济高质量发展体制机制
- 构建支持全面创新体制机制
- 完善城乡融合发展体制机制
- 完善高水平对外开放体制机制

### 4.2 2024年中央经济工作会议

**九大重点任务**：
1. 大力提振消费、提高投资效益
2. 以科技创新引领新质生产力发展
3. 发展现代化产业体系
4. 统筹推进新型城镇化和乡村全面振兴
5. 扩大高水平对外开放
6. 有效防范化解重点领域风险
7. 加大保障和改善民生力度
8. 加强生态文明建设
9. 维护社会和谐稳定

### 4.3 2025年全国两会（政府工作报告）

**GDP目标**：5%左右
**重点**：
- 稳增长、促改革、调结构、惠民生、防风险
- 因地制宜发展新质生产力
- 扩大国内需求
- 推动高水平科技自立自强
