# 言语理解与表达核心知识点

> 言语理解是行测中题量最大的模块之一（京考约35题），也是最容易通过技巧提分的模块。

---

## 一、主旨概括题（⭐⭐⭐⭐⭐）

### 1.1 解题核心

**找主旨句**：文段的核心观点/中心思想

### 1.2 常见文段结构

| 结构 | 特征 | 主旨位置 |
|------|------|----------|
| 总分 | 先提出观点，再展开论述 | 首句/首段 |
| 分总 | 先铺垫/举例，再总结 | 尾句 |
| 总分总 | 提出-论证-重申 | 首尾呼应，看尾句 |
| 转折 | 虽然…但是…/然而… | 转折之后 |
| 因果 | 因此/所以/可见/由此 | 结论句 |
| 并列 | A…B…/一方面…另一方面… | 全面概括 |

### 1.3 关键词标志

| 标志词 | 作用 | 主旨方向 |
|--------|------|----------|
| 但是/然而/却/不过 | 转折 | 转折后 |
| 因此/所以/可见/总之 | 总结 | 总结句 |
| 换言之/也就是说 | 同义替换 | 替换后的表述 |
| 关键是/重要的是/核心在于 | 强调 | 强调内容 |
| 不仅…而且/更 | 递进 | 递进后（更重要的） |

### 1.4 错误选项特征

| 类型 | 表现 |
|------|------|
| 偷换概念 | 将文段中的A概念换成了B |
| 以偏概全 | 用部分内容代替整体主旨 |
| 过度推断 | 超出文段信息范围 |
| 无中生有 | 文段未提及的内容 |
| 强加因果 | 文段无因果关系却强行建立 |

## 二、意图判断题（⭐⭐⭐⭐⭐）

### 2.1 与主旨题的区别

- 主旨题：问"这段文字主要说明/意在强调"→ 找中心句
- 意图题：问"这段文字意在/旨在"→ 可能需要推一步

### 2.2 判断标准

| 文段类型 | 选什么 |
|----------|--------|
| 有明确观点 | 选观点本身（同主旨） |
| 提出问题但无对策 | 选针对问题的对策 |
| 讲故事/举例子 | 选背后的道理/启示 |

---

## 三、细节判断题（⭐⭐⭐⭐）

### 3.1 解题方法

逐项对照原文，找"与原文不符/相符"的选项。

### 3.2 常见设错方式

| 设错类型 | 说明 | 识别方法 |
|----------|------|----------|
| 偷换时态 | "已经"→"将要" | 注意时间词 |
| 偷换数量 | "部分"→"全部" | 注意量词 |
| 偷换逻辑 | "可能"→"一定" | 注意语气词 |
| 张冠李戴 | A的特征说成B的 | 注意主语 |
| 无中生有 | 原文未提及 | 回原文找依据 |
| 曲解文意 | 歪曲原文含义 | 精确理解原文 |

---

## 四、逻辑填空题（⭐⭐⭐⭐⭐）

### 4.1 解题思路

```
1. 分析语境 → 找逻辑关系（转折/递进/并列/解释）
2. 确定空缺处语义方向
3. 辨析选项词语差异
4. 代入验证
```

### 4.2 语境分析

| 逻辑关系 | 标志词 | 空缺语义 |
|----------|--------|----------|
| 转折 | 但是/然而/却 | 与前文相反 |
| 递进 | 甚至/更/不仅…而且 | 程度加深 |
| 并列 | 同时/也/既…又 | 语义相近 |
| 解释 | 即/也就是说/换言之 | 同义互解 |
| 对应 | 冒号、破折号后 | 前后对应 |

### 4.3 词语辨析维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 语义轻重 | 程度差异 | 批评 < 批判 < 抨击 |
| 适用范围 | 搭配对象不同 | 侵犯（权利）vs 侵占（土地） |
| 感情色彩 | 褒/贬/中性 | 果断（褒）vs 武断（贬） |
| 语体色彩 | 书面/口语 | 商榷（书面）vs 商量（口语） |
| 固定搭配 | 习惯用法 | 提高水平、改善条件、增强意识 |

### 4.4 高频易混词语

| 词组 | 区别 |
|------|------|
| 截然不同 / 大相径庭 | 截然不同：完全不同；大相径庭：相差很远 |
| 南辕北辙 / 背道而驰 | 南辕北辙：行动与目的相反；背道而驰：方向相反 |
| 耳濡目染 / 潜移默化 | 耳濡目染：强调亲身感受；潜移默化：强调不知不觉 |
| 不以为然 / 不以为意 | 不以为然：不认为对；不以为意：不放在心上 |
| 无可厚非 / 无可非议 | 无可厚非：不必过分指责；无可非议：完全正确 |
| 鱼龙混杂 / 良莠不齐 | 鱼龙混杂：好坏混在一起（物）；良莠不齐：好人坏人都有（人） |
| 举足轻重 / 举重若轻 | 举足轻重：地位重要；举重若轻：能力强，轻松应对 |
| 固步自封 / 墨守成规 | 固步自封：不求进步；墨守成规：按老规矩办事 |
| 相辅相成 / 相得益彰 | 相辅相成：互相配合；相得益彰：互相配合使优点更突出 |
| 未雨绸缪 / 防患未然 | 未雨绸缪：事先准备；防患未然：预防灾祸 |

## 五、语句排序题（⭐⭐⭐）

### 5.1 解题步骤

```
1. 看选项首句 → 排除不适合做首句的
2. 找关联词配对 → 确定相邻句
3. 找指代词 → 确定前后顺序
4. 看逻辑顺序 → 时间/空间/递进
5. 代入验证
```

### 5.2 不适合做首句的

- 以"这""那""其""此"等指代词开头
- 以"但是""然而""因此"等关联词开头
- 补充说明性的句子

### 5.3 确定相邻的标志

- 关联词配对（虽然…但是…）
- 指代词与其指代对象
- 话题一致性

## 六、语句衔接题（⭐⭐⭐）

### 解题方法

1. **看位置**：句首（总领）、句中（承上启下）、句尾（总结）
2. **看话题**：前后话题一致
3. **看逻辑**：与上下文逻辑关系匹配
4. **看句式**：与前后句式保持一致（对仗/排比）

---

## 七、篇章阅读（京考特色 ⭐⭐⭐⭐）

### 7.1 京考特点

- 给出一篇800-1200字的文章
- 设置5道小题
- 题型涵盖：主旨、细节、词语理解、标题、推断

### 7.2 解题策略

1. **先看题目**：明确5道题分别问什么
2. **快速通读**：把握文章结构和主旨
3. **定位作答**：根据题目关键词定位到具体段落
4. **注意顺序**：题目顺序通常与文章段落顺序一致

## 八、高频成语积累（50个必背）

| 成语 | 含义 | 易错点 |
|------|------|--------|
| 首当其冲 | 最先受到攻击或遭遇灾难 | ≠首先、第一个 |
| 差强人意 | 大体上还能使人满意 | ≠不如人意 |
| 万人空巷 | 形容盛况，人们都出来了 | ≠街上没人 |
| 望其项背 | 能够赶得上 | 常用否定形式"难以望其项背" |
| 炙手可热 | 权势大，气焰盛 | ≠热门、受欢迎 |
| 目无全牛 | 技艺纯熟 | ≠没有全局观 |
| 文不加点 | 写文章一气呵成 | ≠不加标点 |
| 不刊之论 | 不可改动的言论（正确） | ≠不能刊登 |
| 久假不归 | 长期借用不归还 | ≠长期请假 |
| 危言危行 | 正直的言行 | ≠危险的言行 |
