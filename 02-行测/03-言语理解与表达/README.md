# 2.3 言语理解与表达

> 京考言语最大特点：以篇章阅读为主，无选词填空。

---

## 考查内容

言语理解与表达主要测查运用语言文字进行思考和交流、迅速准确地理解和把握文字材料内涵的能力。

---

## 题型分布

| 题型 | 题量 | 说明 |
|------|------|------|
| 篇章阅读 | 约24题（6篇×4题） | 京考核心题型 |
| 片段阅读 | 约6题 | 中心理解、细节判断 |
| 语句表达 | 约2-4题 | 语句排序、语句填充 |

> ⚠️ 京考言语**没有逻辑填空/选词填空**，这是与国考最大的区别。

---

## 篇章阅读攻略

### 题型分类
1. **主旨概括题** — 文章/段落的中心思想
2. **细节判断题** — 与原文是否一致
3. **词句理解题** — 特定词语/句子的含义
4. **推断题** — 根据文章推断作者观点
5. **标题选择题** — 为文章选择合适标题

### 解题步骤
1. **先看题目** — 明确问什么
2. **快速通读** — 把握文章结构和主旨
3. **定位原文** — 找到题目对应的段落
4. **比对选项** — 排除偷换概念、无中生有、过度推断

### 常见陷阱
- 偷换概念：将原文的A偷换为近似的B
- 无中生有：选项内容原文未提及
- 过度推断：超出原文信息范围
- 以偏概全：用局部信息代替整体结论

## 典型例题精讲

### 例题1：篇章阅读 — 主旨概括

**材料**（节选）：

> 数字技术正在深刻改变城市治理的方式。北京市通过建设"城市大脑"，将交通、环境、公共安全等领域的数据汇聚整合，实现了从"经验决策"到"数据决策"的转变。例如，通过实时监测交通流量，智能调控信号灯时长，使重点路段通行效率提升了15%。同时，利用卫星遥感和物联网技术，对城市热岛效应、空气质量等进行动态监测，为精准施策提供科学依据。然而，数字化治理也面临数据孤岛、隐私保护、技术门槛等挑战，需要在推进过程中不断完善制度保障。

**问题**：这段文字主要说明的是：

A. 北京"城市大脑"建设取得了显著成效  
B. 数字技术为城市治理带来了机遇与挑战  
C. 数据决策已经完全取代了经验决策  
D. 数字化治理面临的问题难以解决

**答案**：B

**解析**：
- 文段结构：总（数字技术改变城市治理）→ 分（举例说明成效）→ 转折（然而…面临挑战）
- 转折前讲机遇/成效，转折后讲挑战
- 全面概括 = 机遇 + 挑战 → 选B
- A只涉及前半部分，以偏概全
- C"完全取代"过于绝对，原文说"从…到…的转变"
- D"难以解决"无中生有，原文说"需要不断完善"

**技巧**：有转折的文段，主旨往往是对转折前后的全面概括，而非只取一端。

### 例题2：篇章阅读 — 细节判断

**问题**（接上文材料）：根据这段文字，下列说法正确的是：

A. 北京所有路段的通行效率都提升了15%  
B. "城市大脑"只用于交通管理领域  
C. 数字化治理需要制度保障的配合  
D. 物联网技术主要用于调控交通信号灯

**答案**：C

**解析**：
- A错误：原文是"重点路段"，不是"所有路段"（以偏概全）
- B错误：原文提到"交通、环境、公共安全等领域"（缩小范围）
- C正确：原文"需要在推进过程中不断完善制度保障"
- D错误：物联网用于"城市热岛效应、空气质量等动态监测"（偷换对象）

**技巧**：细节判断题逐项比对原文，注意"所有/全部""只/仅""主要"等限定词。

### 例题3：语句排序

**题目**：将以下句子排列成语意连贯的段落：

①因此，保护生物多样性已成为全球共识  
②生物多样性是人类赖以生存的基础  
③然而，由于人类活动的影响，全球物种正以前所未有的速度消失  
④它为人类提供食物、药物、清洁的空气和水等生态服务  
⑤各国纷纷采取建立自然保护区、制定法律法规等措施加以应对

A. ②④③①⑤  
B. ②③④①⑤  
C. ③②④①⑤  
D. ①②④③⑤

**答案**：A

**解析**：
- ②是总起句（提出"生物多样性是基础"）
- ④紧跟②，具体解释"基础"体现在哪里（提供生态服务）
- ③转折，指出面临的威胁
- ①"因此"总结，得出保护的必要性
- ⑤具体措施

逻辑链：提出概念 → 解释重要性 → 转折指出问题 → 总结共识 → 具体行动

**技巧**：
1. 找首句（通常是总起/背景，排除以"因此""然而"开头的）
2. 找关联词配对（"然而"前应有正面内容，"因此"前应有原因）
3. 验证逻辑通顺

## 言语理解高频易错点

### 篇章阅读常见错误选项类型

| 类型 | 特征 | 识别方法 |
|------|------|----------|
| 偷换概念 | 将A换成近似的B | 比对原文关键词是否一致 |
| 无中生有 | 原文未提及的内容 | 检查选项信息是否有出处 |
| 过度推断 | 超出原文范围的结论 | 原文是否支持这个推论 |
| 以偏概全 | 局部信息代替整体 | 注意"部分/全部""有些/所有" |
| 绝对化表述 | "一定""必然""完全" | 原文是否有如此绝对的表述 |
| 因果倒置 | 颠倒原因和结果 | 检查逻辑方向是否正确 |
| 时态混淆 | 将来时说成已完成 | 注意"将""已""正在"等时态词 |

### 主旨题解题口诀

```
转折之后是重点，
因果结论看后面，
并列结构要全面，
总分结构看总句。
```
