# 2.3 言语理解与表达

> 京考言语最大特点：以篇章阅读为主，无选词填空。

---

## 考查内容

言语理解与表达主要测查运用语言文字进行思考和交流、迅速准确地理解和把握文字材料内涵的能力。

---

## 题型分布

| 题型 | 题量 | 说明 |
|------|------|------|
| 篇章阅读 | 约24题（6篇×4题） | 京考核心题型 |
| 片段阅读 | 约6题 | 中心理解、细节判断 |
| 语句表达 | 约2-4题 | 语句排序、语句填充 |

> ⚠️ 京考言语**没有逻辑填空/选词填空**，这是与国考最大的区别。

---

## 篇章阅读攻略

### 题型分类
1. **主旨概括题** — 文章/段落的中心思想
2. **细节判断题** — 与原文是否一致
3. **词句理解题** — 特定词语/句子的含义
4. **推断题** — 根据文章推断作者观点
5. **标题选择题** — 为文章选择合适标题

### 解题步骤
1. **先看题目** — 明确问什么
2. **快速通读** — 把握文章结构和主旨
3. **定位原文** — 找到题目对应的段落
4. **比对选项** — 排除偷换概念、无中生有、过度推断

### 常见陷阱
- 偷换概念：将原文的A偷换为近似的B
- 无中生有：选项内容原文未提及
- 过度推断：超出原文信息范围
- 以偏概全：用局部信息代替整体结论

---

## 备考建议

| 阶段 | 方法 |
|------|------|
| 基础期 | 每天精读1-2篇真题文章，分析文章结构 |
| 提升期 | 限时训练，每篇文章控制在5分钟内 |
| 冲刺期 | 整套模拟，训练篇章间的切换速度 |

### 日常积累
- 阅读《人民日报》评论员文章
- 关注《半月谈》《瞭望》等时政期刊
- 培养快速抓取文章主旨的习惯
