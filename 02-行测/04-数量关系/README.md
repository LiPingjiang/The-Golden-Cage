## 数字推理

### 常见规律类型
- 等差数列、等比数列
- 和数列、积数列
- 多次方数列
- 分数数列
- 递推数列
- 组合数列（交叉、分组）

### 解题思路
1. 先观察相邻项之间的差
2. 再看比值关系
3. 考虑多次方变形
4. 注意分组或交叉规律

## 备考策略

### 核心原则
> 只学6-8类基础高频题型，复杂题型不深究，难题直接放弃。

### 推荐学习顺序
1. 工程问题（赋值法）
2. 利润问题（公式法）
3. 行程问题（基础追及、相遇）
4. 容斥问题（公式+画图）
5. 排列组合（基础计数原理）
6. 概率问题（古典概型）

### 考场策略
- 数量关系放在最后做
- 先扫一遍，挑出能做的简单题
- 每题控制在1.5分钟内
- 完全没思路的直接猜C或D

## 典型例题精讲

### 例题1：工程问题（赋值法）

**题目**：一项工程由甲、乙两个工程队合作完成需要20天，甲队单独完成需要30天。如果乙队单独完成这项工程，需要多少天？

A. 45天　　B. 50天　　C. 55天　　D. 60天

**解析**：
- 赋值工程总量为60（30和20的最小公倍数）
- 甲的效率 = 60 ÷ 30 = 2/天
- 甲乙合作效率 = 60 ÷ 20 = 3/天
- 乙的效率 = 3 - 2 = 1/天
- 乙单独完成时间 = 60 ÷ 1 = **60天**

**答案**：D

**方法**：赋值法 — 赋值工程总量为各时间的最小公倍数

---

### 例题2：工程问题（交替合作）

**题目**：甲单独完成一项工程需12天，乙单独完成需18天。如果甲先做1天，然后乙做1天，再甲做1天……如此交替进行，完成这项工程共需多少天？

A. 13天　　B. 14天　　C. 14.5天　　D. 15天

**解析**：
- 赋值总量 = 36（12和18的最小公倍数）
- 甲效率 = 36÷12 = 3/天，乙效率 = 36÷18 = 2/天
- 每2天（一个周期）完成 3+2 = 5
- 36 ÷ 5 = 7余1，即7个完整周期（14天）完成35，还剩1
- 第15天轮到甲做，甲效率3 > 剩余1，所以甲不到1天就完成
- 但注意：第15天甲开始做，当天即可完成

**答案**：D（共需15天，第15天甲工作1/3天即完成，但按天计算为15天）

等等，重新计算：7个周期=14天完成35，第15天甲做，完成剩余1，所以共**15天**。

**答案**：D

---

### 例题3：行程问题（相遇）

**题目**：甲乙两人分别从A、B两地同时出发相向而行，甲的速度为60km/h，乙的速度为40km/h。两人在距A地180km处相遇，则A、B两地相距多少公里？

A. 240　　B. 280　　C. 300　　D. 320

**解析**：
- 相遇时甲走了180km
- 甲的时间 = 180 ÷ 60 = 3小时
- 同时出发同时相遇，乙也走了3小时
- 乙走的距离 = 40 × 3 = 120km
- AB总距离 = 180 + 120 = **300km**

**答案**：C

**方法**：相遇问题核心公式 — 总路程 = (v₁ + v₂) × 时间

### 例题4：利润问题

**题目**：某商品进价为200元，标价为360元，打八折销售后利润率为多少？

A. 36%　　B. 40%　　C. 44%　　D. 48%

**解析**：
- 售价 = 360 × 80% = 288元
- 利润 = 288 - 200 = 88元
- 利润率 = 利润 ÷ 进价 = 88 ÷ 200 = **44%**

**答案**：C

**公式**：利润率 =（售价 - 成本）÷ 成本 × 100%

---

### 例题5：容斥问题

**题目**：某班50人，参加数学兴趣小组的有30人，参加语文兴趣小组的有25人，两个小组都没参加的有5人。则两个小组都参加的有多少人？

A. 8人　　B. 10人　　C. 12人　　D. 15人

**解析**：
两集合容斥公式：A∪B = A + B - A∩B

参加至少一个小组的人数 = 50 - 5 = 45人
45 = 30 + 25 - A∩B
A∩B = 55 - 45 = **10人**

**答案**：B

**公式**：|A∪B| = |A| + |B| - |A∩B|

---

### 例题6：数字推理

**题目**：2, 5, 11, 23, 47, ( )

A. 90　　B. 93　　C. 95　　D. 97

**解析**：
观察相邻项关系：
- 5 = 2×2 + 1
- 11 = 5×2 + 1
- 23 = 11×2 + 1
- 47 = 23×2 + 1
- 下一项 = 47×2 + 1 = **95**

**答案**：C

**规律**：递推数列，每项 = 前一项 × 2 + 1

## 数量关系常用公式速查表

| 题型 | 核心公式 |
|------|----------|
| 工程问题 | 工作总量 = 效率 × 时间 |
| 行程问题 | 路程 = 速度 × 时间 |
| 相遇问题 | 总路程 = (v₁+v₂) × t |
| 追及问题 | 路程差 = (v₁-v₂) × t |
| 利润问题 | 利润率 = (售价-成本)/成本 |
| 浓度问题 | 浓度 = 溶质/溶液 |
| 容斥（两集合） | A∪B = A+B-A∩B |
| 容斥（三集合） | A∪B∪C = A+B+C-A∩B-A∩C-B∩C+A∩B∩C |
| 排列 | A(n,m) = n!/(n-m)! |
| 组合 | C(n,m) = n!/[m!(n-m)!] |
| 等差数列求和 | S = n(a₁+aₙ)/2 |
| 等比数列求和 | S = a₁(1-qⁿ)/(1-q) |
