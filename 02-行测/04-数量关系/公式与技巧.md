# 数量关系核心公式与解题技巧

> 数量关系是行测中难度最高但也最有区分度的模块。掌握高频题型公式，考场上"会做的全对、不会的果断跳过"。

---

## 一、工程问题（高频 ⭐⭐⭐⭐⭐）

### 1.1 基本公式

```
工作总量 = 工作效率 × 工作时间
```

### 1.2 常用方法

| 方法 | 适用场景 | 操作 |
|------|----------|------|
| 赋值法 | 已知时间求效率 | 设工作总量为时间的最小公倍数 |
| 特值法 | 只有比例关系 | 设某个量为特值（如1或100） |

### 1.3 典型题型

**合作问题**：
- 甲单独做a天完成，乙单独做b天完成
- 合作效率 = 1/a + 1/b
- 合作时间 = 1 ÷ (1/a + 1/b) = ab/(a+b)

**交替合作**：按周期计算，注意最后不满一个周期的情况

**水管问题**：注水为正效率，放水为负效率

## 二、行程问题（高频 ⭐⭐⭐⭐⭐）

### 2.1 基本公式

```
路程 = 速度 × 时间
```

### 2.2 常见模型

| 模型 | 核心关系 |
|------|----------|
| 相遇问题 | 路程和 = (v₁ + v₂) × t |
| 追及问题 | 路程差 = (v₁ - v₂) × t |
| 流水行船 | 顺流速度 = 船速 + 水速；逆流速度 = 船速 - 水速 |
| 环形追及 | 追及一圈的时间 = 周长 ÷ 速度差 |
| 往返问题 | 平均速度 = 2v₁v₂/(v₁+v₂)（≠算术平均） |

### 2.3 比例法

当路程相同时：时间比 = 速度的反比  
当时间相同时：路程比 = 速度比

## 三、利润问题（⭐⭐⭐⭐）

### 3.1 基本公式

```
利润 = 售价 - 成本
利润率 = 利润 ÷ 成本 × 100%
售价 = 成本 ×（1 + 利润率）
折扣价 = 定价 × 折扣率
```

### 3.2 常见设置

- 打折后仍有利润：定价 × 折扣 > 成本
- 多种商品混合利润：加权平均

---

## 四、排列组合与概率（⭐⭐⭐⭐）

### 4.1 基本公式

| 公式 | 含义 |
|------|------|
| A(n,m) = n!/(n-m)! | 从n个中取m个排列（有序） |
| C(n,m) = n!/[m!(n-m)!] | 从n个中取m个组合（无序） |

### 4.2 常用方法

| 方法 | 适用场景 |
|------|----------|
| 捆绑法 | 相邻问题：把必须相邻的元素捆绑为一个整体 |
| 插空法 | 不相邻问题：先排其他元素，再在空隙中插入 |
| 隔板法 | 相同元素分组：n个相同球放入m个不同盒，C(n-1, m-1) |
| 定位法 | 某元素有特殊位置要求：先安排受限元素 |

### 4.3 概率

```
概率 = 满足条件的情况数 ÷ 总情况数
```

## 五、容斥原理（⭐⭐⭐⭐）

### 5.1 两集合

```
A∪B = A + B - A∩B
```

### 5.2 三集合（标准公式）

```
A∪B∪C = A + B + C - A∩B - A∩C - B∩C + A∩B∩C
```

### 5.3 三集合（非标准/画图法）

当题目给出"只满足两个条件"或"至少满足两个条件"时，画韦恩图更直观。

## 六、数字推理（⭐⭐⭐）

### 6.1 常见规律

| 规律类型 | 特征 | 示例 |
|----------|------|------|
| 等差数列 | 相邻两项差相等 | 2, 5, 8, 11, 14 |
| 等比数列 | 相邻两项比相等 | 2, 6, 18, 54 |
| 二级差数列 | 差组成等差/等比 | 1, 2, 4, 7, 11（差为1,2,3,4） |
| 递推数列 | 前两项运算得后一项 | 1, 1, 2, 3, 5, 8（斐波那契） |
| 幂次数列 | 与n²、n³相关 | 1, 4, 9, 16, 25（n²） |
| 分组数列 | 奇数项/偶数项各成规律 | |
| 多重数列 | 间隔取项各成规律 | |

### 6.2 解题思路

```
1. 看趋势：递增/递减/波动
2. 看幅度：变化平缓→做差；变化剧烈→做商/看幂次
3. 看特征：都接近某个幂次数→修正数列
```

## 七、几何问题（⭐⭐⭐）

### 7.1 常用面积公式

| 图形 | 面积 |
|------|------|
| 三角形 | S = ½ × 底 × 高 |
| 平行四边形 | S = 底 × 高 |
| 梯形 | S = ½ ×（上底 + 下底）× 高 |
| 圆 | S = πr² |
| 扇形 | S = ½ × r² × θ（弧度）= πr²×(n/360) |

### 7.2 常用体积公式

| 立体 | 体积 |
|------|------|
| 长方体 | V = 长 × 宽 × 高 |
| 圆柱 | V = πr²h |
| 圆锥 | V = ⅓πr²h |
| 球 | V = 4/3 × πr³ |

### 7.3 几何技巧

- **相似三角形**：对应边成比例，面积比 = 边长比的平方
- **勾股定理**：a² + b² = c²（常见勾股数：3-4-5, 5-12-13, 8-15-17）
- **圆的性质**：直径所对圆周角为90°

## 八、其他高频题型

### 8.1 年龄问题

**核心**：年龄差不变，年龄比在变

### 8.2 日期问题

- 平年365天 = 52周 + 1天
- 闰年366天 = 52周 + 2天
- 闰年判定：能被4整除且不能被100整除，或能被400整除

### 8.3 钟表问题

- 时针速度：0.5°/分钟
- 分针速度：6°/分钟
- 追及速度差：5.5°/分钟

### 8.4 牛吃草问题

```
设每天新长草量为x
原有草量 = (每天消耗量 - x) × 天数
```

## 九、考场策略

| 策略 | 说明 |
|------|------|
| 先易后难 | 工程、利润、容斥优先做 |
| 代入验证 | 选项为具体数值时，代入验证最快 |
| 整除特性 | 利用整除快速排除选项 |
| 奇偶特性 | 奇±奇=偶，奇±偶=奇 |
| 尾数法 | 只看末位数字排除选项 |
| 放弃策略 | 复杂排列组合、复杂几何果断跳过 |
