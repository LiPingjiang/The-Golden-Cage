# 第5章 专业科目

> 仅公安机关人民警察等特定职位需要加试。以下为可直接用于答题的具体法条与知识点。

## 5.1 哪些职位需要考专业科目

- **公安机关人民警察职位**：必须加试公安专业科目
- 其他职位：部分可能组织专业能力测试（如行政执法岗）

## 5.3 《人民警察法》核心知识（可直接答题）

### 5.3.1 立法目的与基本原则

- **立法目的**：维护国家安全和社会治安秩序，保护公民的合法权益，加强人民警察的队伍建设，从严治警，提高人民警察的素质
- **人民警察节**：每年**1月10日**为中国人民警察节
- **人民警察范围**：包括公安机关、国家安全机关的人民警察，司法行政机关的监狱、戒毒人民警察和人民法院、人民检察院的司法警察

### 5.3.2 公安机关人民警察的14项主要职责（第11条，高频考点）

1. 维护国家安全和社会稳定，防范和打击渗透破坏颠覆分裂活动、恐怖主义、极端主义，防范和处置邪教活动
2. 预防、制止和侦查违法犯罪活动
3. 维护社会治安秩序，制止危害社会治安秩序的行为
4. 保护外国人的合法权益
5. 管理户口、居民身份证和其他基本信息
6. 管理交通安全
7. 管理消防、易燃易爆及危险物品
8. 管理国（境）内外重要人物警卫
9. 管理集会、游行、示威活动
10. 监督管理计算机信息系统的安全保护工作
11. 指导和监督社会治安保卫工作
12. 管理机动车驾驶员的考核发证
13. 管理涉外警务
14. 其他法律法规规定的职责

> **速记方法**：安全稳定→打击犯罪→治安秩序→涉外保护→户籍身份→交通→消防→警卫→集会→网络安全→治保→驾驶考核→涉外警务

### 5.3.3 公安机关人民警察的12项职权（第16条起，高频考点）

| 职权类型 | 具体内容 |
|----------|----------|
| 强制传唤 | 对现场发现的违法犯罪嫌疑人可以强行带离现场，先行拘留 |
| 询问查证 | 对有违法犯罪嫌疑的人员，经出示证件，可以当场查问、检查 |
| 继续盘问 | 对被盘查人员可以继续盘问，最长不超过**24小时**，特殊情况经批准可延长至**48小时** |
| 交通管制 | 发生严重危害社会治安秩序的突发事件时，可以采取现场管制措施 |
| 优先通行 | 执行紧急任务时有优先通行权 |
| 优先使用交通/通信工具 | 遇特殊情况可优先使用交通工具、通信工具、场地和建筑物 |
| 警械武器使用 | 依法使用警械和武器 |

### 5.3.4 人民警察的义务与纪律

**"六不准"（人民警察不得从事的行为，常考多选）**：
1. 不得散布有损党和国家声誉的言论
2. 不得组织或参加非法组织
3. 不得参加旨在反对宪法确定的基本原则的活动
4. 不得参加罢工
5. 不得从事有报酬的经营性活动
6. 不得违反规定收受钱物、牟取不正当利益

**回避制度**：人民警察的**任免、考核、奖惩、警衔晋级**等，直接涉及本人利害关系的，应当回避

**行政处分种类**：警告、记过、记大过、降级、撤职、开除

## 5.4 《治安管理处罚法》核心知识（2025年修订，2026年1月1日起施行）

### 5.4.1 立法目的与基本原则

**立法目的**：维护社会治安秩序，保障公共安全，保护公民、法人和其他组织的合法权益，规范和保障公安机关及其人民警察依法履行治安管理职责

**基本原则**：坚持中国共产党的领导，坚持综合治理；实施治安管理处罚应当公开、公正，尊重和保障人权；坚持教育与处罚相结合

**法律地位**：违反治安管理，尚不够刑事处罚的，给予治安管理处罚（与《刑法》形成衔接，是罪与非罪的重要分水岭）

### 5.4.2 治安管理处罚的种类（第2章，必背）

| 处罚种类 | 具体内容 |
|----------|----------|
| 警告 | 最轻处罚 |
| 罚款 | 金额因违法行为而异 |
| 行政拘留 | 1日以上15日以下；并处的合并执行最长不超过**20日** |
| 吊销许可证 | 针对特定行业资质 |
| 驱逐出境 | 适用于违反治安管理的外国人 |

### 5.4.3 从轻、减轻、不予处罚的情形（高频考点）

**应当从轻或者减轻处罚的情形**：
- 已满14周岁不满18周岁的
- 主动消除或者减轻违法后果，并取得被侵害人谅解的
- 出于他人胁迫或诱骗的
- 主动投案，向公安机关如实陈述自己的违法行为的（**立功**情形）

**不予处罚的情形**：
- 不满**14周岁**的人违反治安管理的，不予处罚，责令其父母或监护人加以管教
- 精神病人在不能辨认或者不能控制自己行为时违反治安管理的，不予处罚
- 违法行为在**6个月内**没有被发现的，不再处罚（追究时效）

**减轻处罚**：
- 已满**14周岁不满16周岁**、已满**16周岁不满18周岁**初次违反治安管理的，可以从轻或者减轻处罚

### 5.4.4 2025修订新增的高频考点（"新法大家学"）

| 新增/修订内容 | 具体条文要点 |
|--------------|--------------|
| 妨害安全驾驶 | 抢夺方向盘、拉扯驾驶员、殴打驾驶员——**实施即违法**，不需造成实际后果 |
| 高空抛物 | 从建筑物或其他高空抛掷物品，只要有危害人身/公共安全之虞，即可处5-10日拘留并罚1000元；情节严重10-15日拘留 |
| 无人机"黑飞" | 违规飞行、升放无人机/气球，情节较重拘5-10日；非法越境拘10-15日 |
| 考试作弊 | 组织、提供器材、出售答案、替考/被替考，处5-15日拘留，并处违法所得1-5倍罚款 |
| 侮辱英烈 | 在公祭场所滋事、污损英烈纪念设施、网络侮辱英烈或美化侵略战争，拘10-15日+5000元罚款 |

> **新法修订总量**：由原来6章119条变为6章**144条**，新增25条、修改60余处

### 5.4.5 治安案件办理时限

| 环节 | 时限 |
|------|------|
| 一般案件办理 | 30日以内（重大复杂可延长30日） |
| 治安拘留复议 | 收到申请之日起2日内决定 |
| 行政复议 | 知道行政行为之日起60日内 |
| 行政诉讼 | 知道或应当知道之日起6个月内 |

## 5.5 《人民警察使用警械和武器条例》核心知识（高频必背）

### 5.5.1 警械分类

| 类型 | 具体警械 |
|------|----------|
| 驱逐性、制服性警械 | 警棍、催泪弹、高压水枪、特种防暴枪等 |
| 约束性警械 | 手铐、脚镣、警绳等 |
| 武器 | 枪支、弹药等致命性警用武器 |

### 5.5.2 可使用驱逐性、制服性警械的8种情形（第7条，经警告无效）

1. 结伙斗殴、殴打他人、寻衅滋事、侮辱妇女或者进行其他流氓活动的
2. 聚众扰乱车站、码头、民用航空站、运动场等公共场所秩序的
3. 非法举行集会、游行、示威的
4. 强行冲越人民警察为履行职责设置的警戒线的
5. 以暴力方法抗拒或者阻碍人民警察依法履行职责的
6. 袭击人民警察的
7. 危害公共安全、社会秩序和公民人身安全的其他行为，需要当场制止的
8. 法律、行政法规规定可以使用警械的其他情形

> **使用限度**：应当以制止违法犯罪行为为限度；当违法犯罪行为得到制止时，应当立即停止使用

### 5.5.3 可使用约束性警械的情形（第8条）

1. 抓获违法犯罪分子或者犯罪重大嫌疑人的
2. 执行逮捕、拘留、看押、押解、审讯、拘传、强制传唤的
3. 法律、行政法规规定可以使用警械的其他情形

> **重要限制**：使用约束性警械**不得故意造成人身伤害**

### 5.5.4 可使用武器的4种紧急情形（第9条，经警告无效）

1. 放火、决水、爆炸等严重危害公共安全的
2. 劫持航空器、船舰、火车、机动车或者驾驶车、船等机动交通工具，故意危害公共安全的
3. 抢夺、抢劫枪支弹药、爆炸、剧毒等危险物品，严重危害公共安全的
4. 使用枪支、爆炸、剧毒等危险物品实施犯罪或者以使用枪支、爆炸、剧毒等危险物品相威胁实施犯罪的

### 5.5.5 违法使用警械武器的法律责任

违法使用警械、武器，造成不应有的人员伤亡、财产损失，构成犯罪的，依法追究**刑事责任**；尚不构成犯罪的，依法给予**行政处分**；对受到伤亡或财产损失的人员，由该人民警察所在公安机关比照国家赔偿的有关规定给予**赔偿**。

## 5.8 典型例题精讲

### 例题1：人民警察法职责

**题目**：根据《中华人民共和国人民警察法》，下列不属于公安机关人民警察职责的是：

A. 维护国家安全和社会稳定  
B. 管理集会、游行、示威活动  
C. 审理刑事案件并作出判决  
D. 管理户口、居民身份证和其他基本信息

**答案**：C

**解析**：审理刑事案件并作出判决是人民法院的职权，不属于公安机关人民警察的14项职责范围。公安机关承担的是侵查、预防、制止违法犯罪等治安管理职责，A、B、D均在《人民警察法》第11条明确列举范围内。

---

### 例题2：警械使用

**题目**：某地一大型庆典活动现场，秦某等4人强行冲越民警设置的警戒线并殴打民警，民警多次警告仍无效。根据《人民警察使用警械和武器条例》，民警可以使用：

A. 警棍　　B. 枪支　　C. 脚镣　　D. 弹药

**答案**：A

**解析**："强行冲越警戒线"和"袭击人民警察"均属于第7条规定的可使用驱逐性、制服性警械的情形，警棍属于此类警械。枪支、弹药属于"武器"，只有在放火、劫持交通工具、抢夺枪支弹药、使用/威胁使用危险物品实施犯罪等4种紧急情形下才可使用，本案不符合使用武器的条件。脚镣属于约束性警械，适用于抓获嫌疑人后的约束，而非制止正在进行的暴力冲撞行为。

---

### 例题3：治安管理处罚法·从宽处理

**题目**：小张（15周岁）与他人斗殴，将对方打伤，事后主动投案并如实陈述违法行为，还赔偿了对方的医疗费用并取得谅解。根据《治安管理处罚法》，公安机关对小张应当：

A. 不予处罚  
B. 从轻或减轻处罚  
C. 从重处罚  
D. 吊销其居民身份证

**答案**：B

**解析**：小张已满14周岁不满18周岁，属于"应当从轻或减轻处罚"的年龄条件之一；同时其"主动消除或减轻违法后果并取得谅解"以及"主动投案如实陈述"均属于法定从轻减轻情节，多个从宽情节叠加，应当从轻或减轻处罚。A错误：不予处罚仅适用于不满14周岁或精神病人不能辨认控制行为的情形。D不属于治安管理处罚种类。
