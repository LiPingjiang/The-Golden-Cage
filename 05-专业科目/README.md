# 第5章 专业科目

> 仅公安机关人民警察等特定职位需要加试。

---

## 5.1 哪些职位需要考专业科目

- **公安机关人民警察职位**：必须加试公安专业科目
- 其他职位：部分可能组织专业能力测试（如行政执法岗）

---

## 5.2 公安专业科目

### 考试概况

| 项目 | 内容 |
|------|------|
| 考试时间 | 公共科目笔试前一天下午 14:00-16:00 |
| 时长 | 120分钟 |
| 满分 | 100分 |
| 题量 | 100道客观题 |
| 题型 | 单选、多选、情境题 |

### 考查内容

| 模块 | 内容 |
|------|------|
| 职业素养 | 政治素质、职业道德、职业纪律 |
| 基础知识 | 法律基础、公安基础知识 |
| 基本能力 | 群众工作、安全防范、信息收集、应急处理 |

### 重点法律

- 人民警察法
- 公安机关组织管理条例
- 治安管理处罚法
- 刑事诉讼法（公安相关部分）
- 行政强制法

---

## 5.3 行政执法岗专业能力测试

部分行政执法类职位会组织专业能力测试，主要考查：

- 法律基础知识
- 执法素质
- 执法能力

> 具体内容以招考简章中各职位的备注说明为准。

---

## 5.4 成绩计算

- 进行专业科目考试的职位：面试成绩和专业科目考试成绩共占50%
- 专业科目考试成绩一般不超过综合成绩的15%
- 具体比重见当年《招考简章》

---

## 5.5 备考建议

### 公安专业科目
- 系统学习公安基础知识教材
- 重点掌握相关法律法规
- 多做情境判断题
- 关注公安工作时事热点

### 行政执法专业测试
- 重点学习行政法相关法律
- 了解执法程序和规范
- 关注执法实务案例
