## 📊 京考速览

| 项目 | 内容 |
|------|------|
| 考试全称 | 北京市各级机关考试录用公务员 |
| 简称 | 京考 |
| 笔试科目 | 行政职业能力测验 + 申论（部分职位加试专业科目） |
| 行测 | 客观题，120分钟，满分100分，约135题 |
| 申论 | 主观题，150分钟，满分100分，3-4题 |
| 报名时间 | 通常每年11月中旬 |
| 笔试时间 | 通常每年12月上旬 |
| 成绩占比 | 笔试50% + 面试50% |
| 报名平台 | [北京市人事考试通用平台](http://rsj.beijing.gov.cn/bm/ztzl/gwy/) |
