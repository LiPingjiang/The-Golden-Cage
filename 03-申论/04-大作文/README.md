# 3.4 大作文

> 分值40分，是拉开申论分差的关键，议论文为主。

---

## 题型特点

- 分值：40分
- 字数：1000-1200字
- 难度：⭐⭐⭐⭐⭐
- 考查能力：文字表达能力 + 综合分析能力

---

## 常见要求

- "请结合给定资料和社会实际，以'……'为题/围绕'……'写一篇议论文"
- "自选角度，自拟题目，写一篇文章"
- 要求：观点明确、论证充分、条理清晰、语言流畅

---

## 文章结构

### 经典五段式

```
第一段：开头（引出话题 + 亮明总论点）  约150字
第二段：分论点一 + 论证                约250字
第三段：分论点二 + 论证                约250字
第四段：分论点三 + 论证                约250字
第五段：结尾（总结升华）              约100-150字
```

### 结构变体

- **递进式**：是什么 → 为什么 → 怎么办
- **并列式**：分论点1 + 分论点2 + 分论点3
- **对比式**：正面论证 + 反面论证 + 总结

---

## 写作方法

### 开头写法

| 方法 | 示例结构 |
|------|----------|
| 概括式 | 背景概述 + 问题引出 + 亮明观点 |
| 转折式 | 成绩/好现象 + 但是 + 问题 + 观点 |
| 引言式 | 名言/政策 + 分析 + 观点 |
| 排比式 | 排比句引出话题 + 观点 |

### 论证方法

| 方法 | 说明 |
|------|------|
| 举例论证 | 用具体事例支撑论点（材料中的+社会热点） |
| 道理论证 | 引用名言、政策、理论 |
| 对比论证 | 正反对比突出论点 |
| 因果论证 | 分析原因和结果的关系 |

### 结尾写法

| 方法 | 示例结构 |
|------|----------|
| 总结式 | 回扣论点 + 展望未来 |
| 呼吁式 | 发出号召 + 美好愿景 |
| 引言式 | 名言收束 + 升华主题 |

---

## 评分档次

| 档次 | 分数 | 标准 |
|------|------|------|
| 一类文 | 35-40分 | 观点鲜明、论证充分、结构严谨、语言优美 |
| 二类文 | 28-34分 | 观点明确、论证较充分、结构完整、语言通顺 |
| 三类文 | 20-27分 | 观点基本明确、论证不够充分、结构基本完整 |
| 四类文 | 20分以下 | 偏题跑题、论证混乱、结构不完整 |

---

## 高分技巧

### 论点提炼
- 从材料主题中提炼，不能脱离材料
- 论点要有高度，体现政策思维
- 分论点之间要有逻辑关系（并列/递进）

### 论据积累
- 政策理论：习近平重要讲话、政府工作报告
- 典型案例：感动中国人物、改革开放成就、北京治理经验
- 名言警句：古今中外经典语句

### 语言提升
- 多用短句，少用长句
- 适当使用排比、对仗
- 避免口语化表达
- 使用政论文语言风格

---

## 常见扣分项

- ❌ 偏题跑题（最致命）
- ❌ 论点不明确
- ❌ 只有论点没有论证
- ❌ 字数不足（每少50字扣1分）
- ❌ 抄袭材料原文大段
- ❌ 错别字（每3个扣1分）
- ❌ 标题缺失（扣2-3分）

## 典型例题精讲

### 例题1：大作文 — 范文赏析

**题目**：请结合给定资料和社会实际，以"让城市更有温度"为题，写一篇议论文。（40分）
要求：观点明确，论证充分，条理清晰，语言流畅，1000-1200字。

**范文**（一类文示例，约1100字）：

---

**让城市更有温度**

城市是人类文明的结晶，承载着人们对美好生活的向往。然而，在快速城镇化进程中，高楼林立的背后是邻里关系的疏离，效率至上的追求中是弱势群体的边缘化，光鲜亮丽的外表下是民生痛点的积累。如何让城市不仅有"高度"和"速度"，更有"温度"，已成为新时代城市治理的核心命题。

让城市更有温度，需要以人为本的治理理念。城市的核心是人，城市治理的出发点和落脚点都应是增进民生福祉。北京市推行的"接诉即办"改革，以12345市民服务热线为纽带，将群众诉求作为治理的"指挥棒"，实现了从"政府端菜"到"群众点菜"的转变。五年来，响应率达到100%，解决率从53%提升至93%以上，一条热线串起了万家民生。这启示我们，城市的温度来自对每一个普通人需求的尊重和回应，来自"民有所呼、我有所应"的治理自觉。

让城市更有温度，需要精细化的服务供给。城市管理应该像绣花一样精细，在细微处见真章。老旧小区加装电梯，解决的是老年人"上下楼难"的切身之痛；社区食堂的开设，温暖的是独居老人的一日三餐；口袋公园的建设，填补的是居民身边的绿色空间。这些看似微小的改变，却实实在在提升了市民的获得感和幸福感。精细化不是面面俱到的平均用力，而是精准识别需求、靶向解决问题，把有限的资源用在群众最需要的地方。

让城市更有温度，需要多元共治的参与格局。一座有温度的城市，不能仅靠政府"独唱"，更需要社会各方"合唱"。社区"小院议事厅"让居民从旁观者变为参与者，志愿服务让邻里守望相助成为常态，社会组织的介入让专业力量下沉基层。当每个人都成为城市温度的传递者，当共建共治共享成为城市治理的底色，城市的温度就不再是自上而下的"供暖"，而是由内而外的"自发热"。

当然，让城市更有温度并非一蹴而就。它需要制度的持续完善——用法治保障公平正义；需要技术的善意运用——让数字化服务弥合而非加剧鸿沟；需要文化的涵养浸润——让包容、友善成为城市精神的底色。唯有将"温度"融入城市规划、建设、管理的全过程，才能让每一位市民都感受到城市的善意。

城市让生活更美好，而温度让城市更美好。站在新时代的起点上，让我们以人民为中心，以精细化治理为路径，以共建共治为动力，共同建设一座既有现代化品质、又有人情味温度的城市，让每一个生活在其中的人都能感受到被关怀、被尊重、被温暖。

---

**点评**：
- ✅ 论点鲜明：开头明确提出"让城市有温度"的命题
- ✅ 三个分论点递进：理念→服务→参与
- ✅ 论证充实：每段有具体案例（接诉即办、加装电梯、小院议事厅）
- ✅ 语言规范：政论文风格，有排比和对仗
- ✅ 结尾升华：从城市温度上升到人民幸福
