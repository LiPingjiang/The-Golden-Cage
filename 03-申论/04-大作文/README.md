# 3.4 大作文

> 分值40分，是拉开申论分差的关键，议论文为主。

---

## 题型特点

- 分值：40分
- 字数：1000-1200字
- 难度：⭐⭐⭐⭐⭐
- 考查能力：文字表达能力 + 综合分析能力

---

## 常见要求

- "请结合给定资料和社会实际，以'……'为题/围绕'……'写一篇议论文"
- "自选角度，自拟题目，写一篇文章"
- 要求：观点明确、论证充分、条理清晰、语言流畅

---

## 文章结构

### 经典五段式

```
第一段：开头（引出话题 + 亮明总论点）  约150字
第二段：分论点一 + 论证                约250字
第三段：分论点二 + 论证                约250字
第四段：分论点三 + 论证                约250字
第五段：结尾（总结升华）              约100-150字
```

### 结构变体

- **递进式**：是什么 → 为什么 → 怎么办
- **并列式**：分论点1 + 分论点2 + 分论点3
- **对比式**：正面论证 + 反面论证 + 总结

---

## 写作方法

### 开头写法

| 方法 | 示例结构 |
|------|----------|
| 概括式 | 背景概述 + 问题引出 + 亮明观点 |
| 转折式 | 成绩/好现象 + 但是 + 问题 + 观点 |
| 引言式 | 名言/政策 + 分析 + 观点 |
| 排比式 | 排比句引出话题 + 观点 |

### 论证方法

| 方法 | 说明 |
|------|------|
| 举例论证 | 用具体事例支撑论点（材料中的+社会热点） |
| 道理论证 | 引用名言、政策、理论 |
| 对比论证 | 正反对比突出论点 |
| 因果论证 | 分析原因和结果的关系 |

### 结尾写法

| 方法 | 示例结构 |
|------|----------|
| 总结式 | 回扣论点 + 展望未来 |
| 呼吁式 | 发出号召 + 美好愿景 |
| 引言式 | 名言收束 + 升华主题 |

---

## 评分档次

| 档次 | 分数 | 标准 |
|------|------|------|
| 一类文 | 35-40分 | 观点鲜明、论证充分、结构严谨、语言优美 |
| 二类文 | 28-34分 | 观点明确、论证较充分、结构完整、语言通顺 |
| 三类文 | 20-27分 | 观点基本明确、论证不够充分、结构基本完整 |
| 四类文 | 20分以下 | 偏题跑题、论证混乱、结构不完整 |

---

## 高分技巧

### 论点提炼
- 从材料主题中提炼，不能脱离材料
- 论点要有高度，体现政策思维
- 分论点之间要有逻辑关系（并列/递进）

### 论据积累
- 政策理论：习近平重要讲话、政府工作报告
- 典型案例：感动中国人物、改革开放成就、北京治理经验
- 名言警句：古今中外经典语句

### 语言提升
- 多用短句，少用长句
- 适当使用排比、对仗
- 避免口语化表达
- 使用政论文语言风格

---

## 常见扣分项

- ❌ 偏题跑题（最致命）
- ❌ 论点不明确
- ❌ 只有论点没有论证
- ❌ 字数不足（每少50字扣1分）
- ❌ 抄袭材料原文大段
- ❌ 错别字（每3个扣1分）
- ❌ 标题缺失（扣2-3分）
