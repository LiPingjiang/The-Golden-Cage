# 3.1 归纳概括

> 申论基础题型，逢考必出，踩点给分是核心。

---

## 题型特点

- 分值：通常15分
- 字数：150-300字
- 难度：⭐⭐（申论中最简单的题型）
- 考查能力：阅读理解能力

---

## 常见问法

- "请根据给定资料X，概括/归纳……的主要问题/原因/做法/经验"
- "请概括给定资料X中……的主要内容"
- "请简述给定资料X反映的主要问题"

---

## 答题方法

### 第一步：审题
1. 明确概括对象（问题/原因/对策/影响/经验）
2. 明确材料范围（给定资料几）
3. 明确字数要求

### 第二步：找点
1. 通读指定材料
2. 标注关键词和关键句
3. 注意转折词（但是、然而）、总结词（因此、总之）、并列词（同时、此外）

### 第三步：加工
1. 同类合并（相同性质的要点合并）
2. 异类分开（不同维度分条列出）
3. 提炼关键词作为每条的"帽子"

### 第四步：书写
1. 总分结构：总括句 + 分条要点
2. 序号标注：一、二、三…… 或 1. 2. 3.
3. 语言精炼：去掉修饰词，保留核心信息

---

## 评分标准

- **踩点给分**：每个要点对应固定分值
- **要点齐全**：宁多勿少，但不能堆砌无关内容
- **条理清晰**：分条作答，逻辑分明
- **语言规范**：不用口语，不抄原文大段

---

## 常见失分原因

| 问题 | 对策 |
|------|------|
| 要点遗漏 | 逐段标注，不放过任何一个要点 |
| 答非所问 | 审题时明确"概括什么" |
| 照抄原文 | 提炼概括，用自己的语言整合 |
| 条理混乱 | 先分类再书写 |
| 超出字数 | 精简语言，删除修饰词 |

---

## 答题模板

```
[总括句]……主要体现在以下几个方面：
一是[关键词]。[具体内容]。
二是[关键词]。[具体内容]。
三是[关键词]。[具体内容]。
……
```
