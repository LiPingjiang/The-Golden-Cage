# 3.1 归纳概括

> 申论基础题型，逢考必出，踩点给分是核心。

---

## 题型特点

- 分值：通常15分
- 字数：150-300字
- 难度：⭐⭐（申论中最简单的题型）
- 考查能力：阅读理解能力

---

## 常见问法

- "请根据给定资料X，概括/归纳……的主要问题/原因/做法/经验"
- "请概括给定资料X中……的主要内容"
- "请简述给定资料X反映的主要问题"

---

## 答题方法

### 第一步：审题
1. 明确概括对象（问题/原因/对策/影响/经验）
2. 明确材料范围（给定资料几）
3. 明确字数要求

### 第二步：找点
1. 通读指定材料
2. 标注关键词和关键句
3. 注意转折词（但是、然而）、总结词（因此、总之）、并列词（同时、此外）

### 第三步：加工
1. 同类合并（相同性质的要点合并）
2. 异类分开（不同维度分条列出）
3. 提炼关键词作为每条的"帽子"

### 第四步：书写
1. 总分结构：总括句 + 分条要点
2. 序号标注：一、二、三…… 或 1. 2. 3.
3. 语言精炼：去掉修饰词，保留核心信息

---

## 评分标准

- **踩点给分**：每个要点对应固定分值
- **要点齐全**：宁多勿少，但不能堆砌无关内容
- **条理清晰**：分条作答，逻辑分明
- **语言规范**：不用口语，不抄原文大段

---

## 常见失分原因

| 问题 | 对策 |
|------|------|
| 要点遗漏 | 逐段标注，不放过任何一个要点 |
| 答非所问 | 审题时明确"概括什么" |
| 照抄原文 | 提炼概括，用自己的语言整合 |
| 条理混乱 | 先分类再书写 |
| 超出字数 | 精简语言，删除修饰词 |

---

## 答题模板

```
[总括句]……主要体现在以下几个方面：
一是[关键词]。[具体内容]。
二是[关键词]。[具体内容]。
三是[关键词]。[具体内容]。
……
```

## 典型例题精讲

### 例题1：归纳概括题

**题目**：请根据给定资料，概括当前我国社区养老服务存在的主要问题。（15分）
要求：全面准确，条理清晰，不超过250字。

**参考答案**：

当前我国社区养老服务主要存在以下问题：一是设施不足。社区养老服务中心覆盖率低，场地面积有限，适老化改造滞后。二是人才短缺。专业护理人员数量不足，薪资待遇低，流动性大，缺乏系统培训。三是服务单一。以基本生活照料为主，医疗康复、精神慰藉、文化娱乐等多元化服务供给不足。四是资金紧张。政府投入有限，社会资本参与积极性不高，运营成本高企。五是供需错位。服务内容与老年人实际需求不匹配，信息化水平低，缺乏精准对接机制。六是标准缺失。服务质量参差不齐，缺乏统一的行业标准和监管体系。

**评分要点**（每点2-3分）：
- ✅ 设施不足
- ✅ 人才短缺
- ✅ 服务单一
- ✅ 资金紧张
- ✅ 供需错位
- ✅ 标准缺失

**答题技巧**：
1. 总分结构，先写总括句
2. 每条用"关键词+具体表现"的格式
3. 要点尽量多（宁多勿少），但不超字数
4. 从材料中提炼，不要自己编造

### 例题2：综合分析题

**题目**：请谈谈你对"城市更新不是大拆大建，而是绣花功夫"这句话的理解。（20分）
要求：分析全面，条理清晰，不超过350字。

**参考答案**：

这句话揭示了城市更新应坚持精细化、渐进式的理念，而非简单粗暴的推倒重来。

具体而言：一方面，"不是大拆大建"意味着城市更新要尊重城市肌理和历史文脉，保留城市记忆和社区网络，避免因过度开发造成文化断裂和社会矛盾。另一方面，"绣花功夫"强调精细化治理：在空间上，注重微更新、微改造，通过小规模渐进式整治提升城市品质；在方法上，注重因地制宜、一区一策，精准回应居民需求；在过程中，注重多方参与、共建共治，让居民成为更新的主体。

当前，北京推行"留改拆"有机结合，实施老旧小区改造、历史文化街区保护等，正是"绣花功夫"的生动实践。因此，城市更新应坚持以人为本、精细治理，在保护中发展、在传承中创新，实现城市的有机更新和可持续发展。

**答题结构**：
1. 点明本质含义（总）
2. 多角度分析（分）：不是什么 + 是什么
3. 联系实际 + 总结（总）

### 例题3：提出对策题

**题目**：假如你是某街道办事处工作人员，请针对辖区内老旧小区物业管理难题，提出改进建议。（25分）
要求：对策具体可行，条理清晰，不超过450字。

**参考答案**：

针对老旧小区物业管理难题，提出以下建议：

一是健全管理机制。推动成立业主委员会，建立业主大会议事规则，明确物业服务标准和收费依据，形成"社区党组织+业委会+物业"三方联动治理格局。

二是引入专业力量。通过政府购买服务、公开招标等方式引入专业物业公司，对暂不具备条件的小区，可由街道托管或引入"准物业"服务，确保基本服务不断档。

三是拓宽资金渠道。积极争取老旧小区改造专项资金，探索住宅专项维修资金使用简化程序，鼓励居民合理分担物业费用，引导社会资本参与停车场、充电桩等配套设施建设运营。

四是推进适老化改造。结合老旧小区综合整治，加装电梯、修缮楼道、增设无障碍设施，改善小区公共环境和基础设施条件。

五是强化居民参与。搭建"小院议事厅"等协商平台，定期开展居民满意度调查，建立问题反馈和快速响应机制，引导居民从"旁观者"变为"参与者"。

六是加强监督考核。建立物业服务质量评价体系，将居民满意度纳入考核指标，对服务不达标的物业企业实行约谈、整改、退出机制。

**评分要点**：对策是否针对性强、可操作、全面、有逻辑层次

### 大作文万能开头模板

**背景引入式**：
> 当前，[领域]正面临[挑战/变革]。[具体现象]日益凸显，[问题]亟待解决。在此背景下，[核心观点]不仅是[意义1]，更是[意义2]。

**名言引入式**：
> [名人]曾说："[名言]"。这一论断深刻揭示了[道理]。当下，[联系现实]，更需要我们[行动方向]。

**对比引入式**：
> 一方面，[正面现象/成就]令人振奋；另一方面，[问题/不足]仍然存在。如何[解决方向]，成为摆在我们面前的重要课题。

### 大作文万能结尾模板

**展望式**：
> [主题]之路任重道远。唯有[做法1]、[做法2]、[做法3]，方能[美好愿景]，为[更大目标]贡献力量。

**呼吁式**：
> 站在新的历史起点上，让我们以[精神/态度]为引领，以[行动]为抓手，[具体做法]，共同书写[领域]的崭新篇章。
