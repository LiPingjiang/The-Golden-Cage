# 3.3 提出对策

> 京考申论高分值题型（25分），要求对策具体、可操作、有针对性。

---

## 题型特点

- 分值：通常25分（京考中分值最高的小题）
- 字数：300-500字
- 难度：⭐⭐⭐
- 考查能力：提出和解决问题能力

---

## 常见问法

- "请根据给定资料，提出解决……问题的对策/建议"
- "假如你是某部门工作人员，请就……提出改进措施"
- "请针对给定资料反映的问题，提出对策建议"

---

## 对策来源

### 1. 直接对策（材料中明确提到的做法）
- 材料中的成功经验
- 专家学者的建议
- 政策文件中的措施

### 2. 间接对策（由问题反推）
- 问题：监管缺失 → 对策：加强监管
- 问题：法律不完善 → 对策：完善法律法规
- 问题：意识薄弱 → 对策：加强宣传教育

---

## 答题方法

### 第一步：找准问题
- 明确要解决什么问题
- 问题的主体是谁
- 问题产生的原因是什么

### 第二步：提出对策
- 针对每个问题/原因提出对应对策
- 对策要具体、可操作
- 注意主体的可行性（政府能做什么、企业能做什么）

### 第三步：组织答案
- 分条列出，每条一个对策方向
- 每条对策结构：关键词 + 具体做法

---

## 对策万能维度

| 维度 | 适用场景 |
|------|----------|
| 完善制度/法规 | 制度缺失、法律空白 |
| 加强监管/执法 | 违规行为、市场乱象 |
| 加大投入/保障 | 资源不足、经费短缺 |
| 强化宣传/教育 | 意识薄弱、认知不足 |
| 创新方式/手段 | 方法陈旧、效率低下 |
| 健全机制/体系 | 体系不完善、协调不畅 |
| 引进人才/技术 | 人才短缺、技术落后 |
| 发挥多方合力 | 单一主体难以解决 |

---

## 答题模板

```
针对……问题，提出以下对策建议：
一是完善……制度。具体而言，[做什么+怎么做+达到什么效果]。
二是加强……监管。[具体措施]。
三是加大……投入。[具体措施]。
四是强化……宣传。[具体措施]。
五是创新……方式。[具体措施]。
```

---

## 高分要点

- ✅ 对策要有针对性（针对具体问题）
- ✅ 对策要可操作（不能太空泛）
- ✅ 对策要全面（多角度、多主体）
- ✅ 对策要有逻辑（从根本到具体）
- ❌ 不能脱离材料空想
- ❌ 不能只说"加强""完善"不说具体怎么做
