# 3.2 综合分析

> 难度较高的题型，要求逻辑严密、层次分明。

---

## 题型特点

- 分值：通常20分（可能出1-2道）
- 字数：200-400字
- 难度：⭐⭐⭐⭐
- 考查能力：综合分析能力

---

## 常见题型

### 1. 词句理解题
- "请谈谈你对'……'这句话的理解"
- 答题结构：解释含义 → 多角度分析 → 总结/对策

### 2. 评价观点题
- "请对'……'这一观点进行评析"
- 答题结构：表明态度 → 分析理由 → 总结

### 3. 启示分析题
- "请分析给定资料X对……有哪些启示"
- 答题结构：总启示 → 分条具体启示

### 4. 原因分析题
- "请分析……的原因"
- 答题结构：直接原因 → 深层原因 → 根本原因

### 5. 比较分析题
- "请比较A和B的异同"
- 答题结构：相同点 → 不同点 → 总结

---

## 答题方法

### 词句理解题三步法
1. **点明本质**：用一句话解释词句的深层含义
2. **多角度展开**：结合材料从不同角度分析
   - 为什么这么说？
   - 有哪些表现？
   - 产生了什么影响？
3. **得出结论**：总结或提出对策

### 评价观点题三步法
1. **亮明观点**：这个观点是正确的/片面的/错误的
2. **论证分析**：
   - 正确在哪里？（结合材料）
   - 不足在哪里？（结合材料）
3. **总结提升**：应该如何正确看待

---

## 答题结构

综合分析题的核心结构是 **"总—分—总"**：

```
[总] 对……的理解/评价是……
[分] 一方面，……。另一方面，……。此外，……。
[总] 因此，……。/综上所述，……。
```

---

## 注意事项

- 综合分析≠简单罗列，要有分析过程
- 观点要从材料中来，不能凭空发挥
- 注意逻辑递进关系
- 结尾要有总结或升华，不能戛然而止
