# 3.2 综合分析

> 难度较高的题型，要求逻辑严密、层次分明。

---

## 题型特点

- 分值：通常20分（可能出1-2道）
- 字数：200-400字
- 难度：⭐⭐⭐⭐
- 考查能力：综合分析能力

---

## 常见题型

### 1. 词句理解题
- "请谈谈你对'……'这句话的理解"
- 答题结构：解释含义 → 多角度分析 → 总结/对策

### 2. 评价观点题
- "请对'……'这一观点进行评析"
- 答题结构：表明态度 → 分析理由 → 总结

### 3. 启示分析题
- "请分析给定资料X对……有哪些启示"
- 答题结构：总启示 → 分条具体启示

### 4. 原因分析题
- "请分析……的原因"
- 答题结构：直接原因 → 深层原因 → 根本原因

### 5. 比较分析题
- "请比较A和B的异同"
- 答题结构：相同点 → 不同点 → 总结

---

## 答题方法

### 词句理解题三步法
1. **点明本质**：用一句话解释词句的深层含义
2. **多角度展开**：结合材料从不同角度分析
   - 为什么这么说？
   - 有哪些表现？
   - 产生了什么影响？
3. **得出结论**：总结或提出对策

### 评价观点题三步法
1. **亮明观点**：这个观点是正确的/片面的/错误的
2. **论证分析**：
   - 正确在哪里？（结合材料）
   - 不足在哪里？（结合材料）
3. **总结提升**：应该如何正确看待

---

## 答题结构

综合分析题的核心结构是 **"总—分—总"**：

```
[总] 对……的理解/评价是……
[分] 一方面，……。另一方面，……。此外，……。
[总] 因此，……。/综上所述，……。
```

---

## 注意事项

- 综合分析≠简单罗列，要有分析过程
- 观点要从材料中来，不能凭空发挥
- 注意逻辑递进关系
- 结尾要有总结或升华，不能戛然而止

## 典型例题精讲

### 例题1：词句理解题

**题目**：请谈谈你对给定资料中"数字鸿沟不仅是技术问题，更是社会公平问题"这句话的理解。（20分）
要求：分析全面，条理清晰，不超过300字。

**参考答案**：

这句话揭示了数字鸿沟的本质不仅在于技术接入的差异，更在于其加剧了社会不平等。

从技术层面看，数字鸿沟表现为不同群体在网络接入、设备拥有、数字技能等方面的差距，老年人、低收入群体、农村居民等面临"用不了、不会用"的困境。

从社会公平层面看，数字鸿沟导致信息获取不对等，使弱势群体在教育、就业、公共服务等方面处于更加不利的地位。数字化服务的普及若忽视这些群体，反而会扩大而非缩小社会差距，形成"强者愈强、弱者愈弱"的马太效应。

因此，弥合数字鸿沟需要超越单纯的技术思维，从制度设计、公共服务供给、数字素养教育等多维度入手，确保数字化发展成果惠及全体人民，实现包容性增长。

**答题结构解析**：
1. **点明本质**（第1段）：解释这句话的深层含义
2. **分层分析**（第2-3段）：技术层面 + 社会公平层面
3. **总结对策**（第4段）：怎么办

### 例题2：评价观点题

**题目**：有人说"基层减负就是减少工作量"，请对此观点进行评析。（20分）
要求：观点明确，分析透彻，不超过350字。

**参考答案**：

这一观点是片面的。基层减负的核心不是简单减少工作量，而是优化工作方式、提升治理效能。

一方面，基层减负确实包含减少不必要的工作负担。如精简会议文件、减少重复报表、清理"指尖上的形式主义"等，让基层干部从繁琐的事务性工作中解脱出来。

另一方面，减负不等于减责、减质。基层减负的根本目的是让干部把更多时间和精力投入到服务群众、解决实际问题中去。减的是形式主义的"虚功"，增的是为民办实事的"实效"。如果简单理解为"少干活"，就偏离了减负的初衷。

更深层看，基层减负需要系统性改革：理顺权责关系，解决"上面千条线、下面一根针"的结构性矛盾；完善考核机制，从"痕迹管理"转向"实绩导向"；推动资源下沉，让基层有权有物有人办事。

因此，基层减负是一项系统工程，既要做好"减法"去除形式主义，更要做好"加法"赋能基层治理。

### 例题3：启示分析题

**题目**：请分析给定资料中某市"社区合伙人"模式对推进基层治理有哪些启示。（20分）
要求：分析全面，条理清晰，不超过300字。

**参考答案**：

"社区合伙人"模式对推进基层治理有以下启示：

一是创新参与机制。通过"合伙人"身份认同，将社会组织、企业、居民等多元主体纳入治理体系，变"政府独唱"为"多方合唱"，激发社会参与活力。

二是整合多方资源。打破政府单一供给模式，引入市场化、专业化力量，实现资源互补、优势叠加，提升公共服务供给效率和质量。

三是建立互利共赢机制。通过场地共享、政策支持、品牌赋能等方式，让参与者既承担社会责任又获得发展空间，形成可持续的合作动力。

四是坚持需求导向。以居民实际需求为出发点匹配"合伙人"，确保服务精准对接，避免供需错位。

五是强化党建引领。在多元参与中发挥基层党组织的统筹协调作用，确保治理方向不偏、公共利益优先。
