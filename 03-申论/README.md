# 第3章 申论

> 申论考的是阅读理解、综合分析、提出和解决问题、文字表达四大能力。

---

## 3.1 考试概况

| 项目 | 内容 |
|------|------|
| 题型 | 主观题 |
| 时间 | 150分钟 |
| 满分 | 100分 |
| 题量 | 3-4道 |
| 作答方式 | 闭卷，一字一格 |

---

## 3.2 题型结构与分值

| 题型 | 题量 | 分值 | 说明 |
|------|------|------|------|
| 归纳概括 | 1道 | 15分 | 基础题型，逢考必出 |
| 综合分析 | 1-2道 | 20分 | 观点评价、原因分析、启示 |
| 提出对策 | 1道 | 25分 | 问题诊断+方案设计 |
| 大作文（议论文） | 1道 | 40分 | 拉开分差的关键 |

---

## 3.3 四大测查能力

根据考试大纲，申论主要测查以下能力：

### 阅读理解能力
- 准确理解给定资料的内容主旨
- 把握各部分之间的关系
- 准确提炼事实所包含的观点

### 综合分析能力
- 对给定资料的全部或部分内容、观点或问题进行分析和归纳
- 多角度地思考资料内容
- 作出合理的推断或评价

### 提出和解决问题能力
- 借助自身的实践经验或生活体验
- 在对给定资料理解分析的基础上
- 发现和界定问题，作出评估或权衡
- 提出解决问题的方案或措施

### 文字表达能力
- 运用说明、陈述、议论等方式
- 准确规范、简明畅达地表述思想观点
- 熟练使用指定的语种

---

## 3.4 京考申论特点

| 特点 | 说明 |
|------|------|
| 材料贴近北京实际 | 常涉及首都治理、城市管理、基层服务等 |
| 不分卷 | 不像国考分副省级/地市级/行政执法 |
| 重视对策 | 提出对策题分值高（25分） |
| 大作文分值大 | 40分，是拉开差距的关键 |
| 格式要求严格 | 一字一格，字数必须合规 |

---

## 3.5 评分标准要点

- **小题**：踩点给分，关键词是核心
- **大作文**：分档评分（一类文35-40分、二类文28-34分、三类文20-27分、四类文20分以下）
- **扣分项**：字数不足、格式错误、偏题跑题、逻辑混乱
