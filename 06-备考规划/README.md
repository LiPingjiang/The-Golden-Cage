# 第6章 备考规划

> 科学的备考规划是上岸的基础，根据自身情况制定个性化方案。

---

## 6.1 备考时间线

以每年12月初笔试为基准，推荐提前4-6个月开始备考。

### 全程规划（6个月版）

| 阶段 | 时间 | 目标 | 重点 |
|------|------|------|------|
| 基础夯实期 | 第1-2个月 | 搭建知识框架 | 系统学习各模块理论 |
| 专项突破期 | 第3-4个月 | 各模块提升 | 分模块刷题+错题复盘 |
| 套题模拟期 | 第5个月 | 适应考试节奏 | 整套限时训练 |
| 冲刺调整期 | 第6个月 | 查漏补缺 | 回顾错题+模拟考试 |

### 短期冲刺（2个月版）

| 阶段 | 时间 | 重点 |
|------|------|------|
| 第1-2周 | 了解考情 | 熟悉题型、做1-2套真题摸底 |
| 第3-4周 | 重点突破 | 资料分析+判断推理+申论小题 |
| 第5-6周 | 全面提升 | 言语+数量+申论大作文 |
| 第7-8周 | 模拟冲刺 | 每周2-3套全真模拟 |

## 6.2 每日时间分配

### 全职备考（每天6-8小时）

| 时段 | 内容 | 时长 |
|------|------|------|
| 早晨 | 政治理论/常识/时政积累 | 30分钟 |
| 上午 | 行测专项训练（资料分析/判断推理） | 2.5小时 |
| 下午 | 行测专项训练（言语/数量） | 2小时 |
| 晚上 | 申论学习与练习 | 2小时 |
| 睡前 | 错题回顾/常识积累 | 30分钟 |

### 在职备考（每天3-4小时）

| 时段 | 内容 | 时长 |
|------|------|------|
| 通勤 | 常识/时政/成语积累（音频/App） | 40分钟 |
| 午休 | 刷10-15道行测题 | 30分钟 |
| 晚上 | 专项训练或申论练习 | 2小时 |
| 周末 | 整套模拟+复盘 | 4-6小时 |

## 6.3 各科目复习优先级

```
🥇 资料分析 — 最先学，每天必练，性价比最高
🥈 判断推理 — 方法性强，掌握方法后提分快
🥉 申论小题 — 踩点给分，方法明确
4️⃣ 言语理解 — 需要长期积累，每天保持阅读
5️⃣ 申论大作文 — 框架+素材积累
6️⃣ 数量关系 — 只学基础题型
7️⃣ 常识判断 — 碎片时间积累
```

---

## 6.4 复习方法论

### 刷题原则
- **精做优于泛做**：10套精做 > 50套泛做
- **真题优先**：真题 > 模拟题
- **限时训练**：从一开始就养成限时习惯
- **错题复盘**：每道错题分析原因，归类整理

### 复盘方法
1. 这道题考的是什么知识点？
2. 我为什么做错？（知识盲区/粗心/方法不对）
3. 正确的解题思路是什么？
4. 同类题目的通用方法是什么？

### 申论练习方法
- 每周至少写1篇完整大作文
- 小题每周练习2-3道
- 写完对照参考答案找差距
- 积累好词好句和政策表述

## 6.5 心态管理

| 问题 | 建议 |
|------|------|
| 焦虑紧张 | 制定计划后按部就班，不与他人比较 |
| 效率低下 | 番茄工作法，25分钟专注+5分钟休息 |
| 遇到瓶颈 | 换个模块学习，或休息一天 |
| 信息焦虑 | 选定1-2个信息源，不要到处看经验帖 |
| 坚持不下去 | 找备考伙伴，互相监督 |

---

## 6.6 考前一周安排

| 天数 | 内容 |
|------|------|
| 考前7天 | 最后一套全真模拟 |
| 考前6天 | 回顾所有错题本 |
| 考前5天 | 复习资料分析公式+申论模板 |
| 考前4天 | 复习判断推理方法+常识高频考点 |
| 考前3天 | 轻度练习，保持手感 |
| 考前2天 | 准备考试用品，熟悉考场路线 |
| 考前1天 | 放松休息，早睡 |
